【传染病学】登革热与基孔肯雅热

发表于2025-09-29|更新于2025-09-29|传染病学
|浏览量:

登革热

登革热（DF） 是由登革病毒（DENV）经过埃及伊蚊及白纹伊蚊传播的自然疫源性疾病，属于乙类传染病。临床特点为突起高热、全身肌肉、骨骼及关节疼痛，疲乏，皮疹，淋巴结肿大，白细胞和血小板减少 。

病原学

  1. 黄病毒科中的黄病毒属，与其他黄病毒属存在交叉免疫；
  2. 根据抗原性不同可分为4个血清型（DENV 1—DENV 4）。各型毒力有所差别，通常3型最强，2、4型次之，1型最弱。

流行病学

  1. 传染源：
    • 患者：在发病前6—18小时至发病后第3天；
    • 隐性感染者
  2. 传播途径：虫媒传播，主要媒介昆虫是埃及伊蚊（东南亚和海南省）及白纹伊蚊（广东、广西、东南亚、太平洋岛屿）。伊蚊是病毒的储存宿主，贮存的时间可超过170天。
  3. 人群易感性：普遍易感，新发传染区以成人居多，地方性疫区以儿童为主。重症患者多发生于地方性流行区的当地居民。
  4. 流行特征：地域性、季节性、周期性。

临床表现

病程分为发热期、极期和恢复期；分为普通型和重症型。普通登革热患者可仅有发热期和恢复期，仅少数患者发展为重症型。

潜伏期：1—14天，多为5—9天。

发热期

  1. 发热、乏力
    • 热型以不规则热为主，少数患者呈双峰热型。
  2. 浅表淋巴结肿大，伴有触痛
  3. 头痛、肌肉痛、关节痛
  4. 皮疹：充血性皮疹、出血性皮疹；
    • 皮岛： 约70%的登革热患者
  5. 不同程度的出血：躲在病程5—8天出现。
  6. 白细胞总数下降。

极期

开始退热，但病情反而加重。

  1. 腹部剧痛、持续呕吐、球结膜水肿、四肢渗漏征、胸腔积液、腹水；
  2. 休克表现，如心动过速、四肢湿冷、脉搏细弱、脉压缩小或测不到血压，进而发生代酸、多功能器官障碍和DIC等；
  3. 皮肤瘀斑和器官出血，以消化道出血、子宫出血和颅内出血多见。

恢复期

实验室与影像学检查

血常规

白细胞总数减少 ，红细胞比容增加， 血小板减少 。

其他检查

  1. 血生化检查：ALT、AST轻中度升高。
  2. 影像学检查：重症登革热患者胸腹部CT及彩超检查可发现胸腔积液、腹水及心包积液等，提示血浆渗漏。

病原学检测

病毒分离、抗原检查（NS1抗原）、特异性抗体检测（IgM和IgG）、特异性核酸检测（RNA）。

诊断

  1. 流行病学史：发病前15天内有登革热流行区或登革热病例地的旅居史；
  2. 临床表现：急起高热、全身多部位疼痛，伴有颜面潮红、皮疹、出血、淋巴结肿大，束臂试验阳性；
  3. 实验室检查：外周血白细胞数和中性粒细胞百分量减少，血小板减少，可有血细胞比容升高，确诊需病原学证据。

诊断标准

  • 疑似病例：流行病学史+登革热临床表现、白细胞血小板减少者
  • 临床诊断病例：疑似病例+登革热特异性IgM抗体阳性/检测出NS1抗原
  • 确诊病例：疑似/临床诊断病例+病毒核酸/分离出登革病毒/恢复期血清特异性IgG抗体阳转或滴度呈4倍以上升高

重症登革热

诊断标准

  1. 严重出血，如皮下血肿、肉眼血尿、咯血、消化道出血、阴道出血及颅内出血等；
  2. 休克，如心动过速、肢端湿冷、脉搏细弱或触不到、脉压减少、血压较病前下降20%或＜90/60mmHg或测不到等；
  3. 严重器官损失，如急性呼吸窘迫综合征或呼吸衰竭、急性心肌炎或心力衰竭、急性严重肝损伤、急性肾功能不全、脑病或脑炎等。

高危人群

老人、婴幼儿和孕妇；伴有糖尿病、高血压、冠心病、消化性溃疡、哮喘、慢性肾病、慢性肝病及免疫缺陷等基础疾病者。

预警指征

  1. 热退后病情恶化或持续高热1周不退；
  2. 严重腹部疼痛，或持续呕吐；
  3. 呼吸急促、胸闷、心悸；
  4. 意识改变，昏睡或烦躁不安；
  5. 明显出血侵向；
  6. 血压下降，少尿；
  7. 早期血小板快速下降；
  8. 血细胞比容升高；
  9. 血清白蛋白降低，心律失常，胸腔积液、腹水等。

鉴别诊断

  1. 急性发热伴出血：肾综合征出血、发热伴血小板减少综合症；
  2. 急性发热伴皮疹：寨卡病毒病、基孔肯雅热、麻疹、风疹、斑疹伤寒、恙虫病等；
  3. 重症登革热应与其他感染性休克、钩端螺旋体病鉴别。

治疗

治疗原则：早发现、早诊断、早防蚊隔离、早治疗。无特效的抗病毒治疗药物，以对症支持治疗为主。

普通登革热

  1. 一般处理：卧床休息，清淡饮食；防蚊隔离（病程超过5天，且热退24小时以上方可结束隔离）；检测指标。
  2. 对症治疗
    • 退热：以物理降温为主，谨慎使用解热镇痛类药物。
    • 补液：以口服补液为主，适当进食流食。
  3. 镇静止痛：安定。

重症登革热

  1. 一般治疗；
  2. 补液：使尿量达到0.5ml/(kg·h)；
  3. 抗休克：先晶后胶，严重出血引起休克者应及时输注红细胞或全血，严重血浆外渗伴有低蛋白血症者应及时输注人血白蛋白；
  4. 防止出血：局部智学输注红细胞、血小板；
  5. 维护重要器官功能。

预防

  1. 管理传染源：隔离、检测检疫。
  2. 切断传播途径：灭蚊
  3. 保护易感人群：蚊帐、驱蚊水、浅色长袖长裤、少到阴暗潮湿角落，登革热疫苗。

基孔肯雅热

基孔肯雅热（CHIK） 是由基孔肯雅病毒（CHIKV）引起，经伊蚊叮咬传播的急性传染病，临床以发热、关节痛或关节炎、皮疹为主要特征。

病原学

基孔肯雅病毒属于披膜病毒科甲病毒属，只有一个血清型。基因组为单股正链RNA，分为3个基因型。

流行病学

  1. 传染源：基孔肯雅热急性期患者、隐性感染者和感染病毒的非人灵长类动物；
  2. 传播途径：虫媒传播，主要为白纹伊蚊和埃及伊蚊；
  3. 易感人群：普遍易感，感染后获得持久免疫力。
  4. 流行特征。

临床表现

潜伏期为1—12天，常为3—7天。

  1. 发热：就诊的主要原因，但不特异。急性起病，发热，体温可达39℃，热程通常1d—7d，可自行退热。少部分病人热退后可再次发热，呈双峰热型。
  2. 关节痛：急性症状期的显著特征。出现时间早，常见对称性多关节受累，远端关节（尤其是腕关节）好发，出现关节肿胀甚至失能性关节痛。
  3. 皮疹：出现时间较登革热早，出血疹叫少见。
  4. 其他皮肤粘膜损伤表现。
  5. 非典型临床症状。

辅助检查

  1. 血常规检查：白细胞计数多为正常；
  2. 生化检查：少数患者ALT、AST、CK升高；
  3. 脑脊液检查：脑膜脑炎患者脑脊液检查复合病毒性损伤的表现；
  4. 病原学检查：
    • 核酸检测：在发病早期（1—7天）通常可以检测病毒RNA；
    • 血清学检测：IgM抗体在发病后3—4天出现，可持续数月；IgG抗体在发病后5—8天输出，可持续数年。

诊断

  1. 流行病学史：发病前12天内曾经到过基孔肯雅热流行区，或居住场所/工作场所周围曾有本病发生；
  2. 临床表现：发热、关节疼痛、皮疹；
  3. 实验室检查：急性期IgM抗体阳性、基孔肯雅热病毒核酸阳性、样本培养分离到病毒、特异性抗体IgG抗体阳转或恢复期抗体增长4倍以上。

诊断标准

  • 疑似病例：存在流行病学史，出现发热、关节疼痛和/或皮疹，并排除登革热和/或其他发热伴出疹性疾病者；
  • 临床诊断病例：疑似病例基础上，急性期IgM抗体阳性者；
  • 确诊病例：疑似病例或临床诊断病例，符合基孔肯雅热病毒核酸阳性、样本培养分离到病毒、特异性抗体IgG抗体阳转或恢复期抗体增长4倍以上任一者。

重症病例

基孔肯雅热的重症病例低于登革热，因其非出血性疾病。以轻症为主，整体预后良好。

临床预警指标

  1. 高热时间超过1周或热退后病情加重；
  2. 神经系统症状：剧烈头痛、嗜睡、烦躁不安等；
  3. 心血管系统症状：心悸、胸闷、胸痛、心律失常、四肢冰冷、尿量减少等；
  4. 呼吸系统症状：气促、呼吸困难等；
  5. 出血表现：皮肤瘀斑、粘膜出血、黑便等。

诊断标准

重症急性病例指经实验室确诊的基孔肯雅热病例，其至少有一处器官或系统出现功能障碍，且病情危及生命，常见以下系统受累：

  • 神经系统受累：脑膜脑炎、驰缓性麻痹、吉兰巴雷综合征、脊髓炎、癫痫、脑神经麻痹等；
  • 心脏受累：心功能衰竭、心源性休克、恶性心律失常等；
  • 急性肝损伤：血清ALT和/或AST＞1000IU/L或总胆红素＞85.5μmol/L。
  • 出凝血障碍：如皮下血肿、呕血、黑便、阴道出血、肉眼血尿颅内出血等严重出血和弥漫性血管内凝血（DIC）等。

治疗

一般治疗

  1. 急性期卧床休息，重点为受损关节制动且避免负重，防止关节损伤加重
  2. 症状严重患者注意监测神志、生命体征、液体入量、尿量、血小板、电
    解质等，密切观察临床预警指标
  3. 避免盲目使用抗菌药物。
  4. 不推荐在疑似或确诊急性非重症基孔肯雅热患者中常规使用皮质类固醇治疗和免疫球蛋白。因原发病正在接受糖皮质激素和免疫球蛋白的人群，根据原发病诊治规范和预期的风险/获益平衡决定。

对症治疗

退热

以物理降温为主，高热、剧烈疼痛不能耐受者可使用对乙酰氨基酚。

镇痛

  1. 急性期关节疼痛不能忍受者可使用对乙酰氨基酚；
  2. 甲氨蝶呤是治疗慢性基孔肯雅热关节炎的最佳改善病情抗风湿药；
