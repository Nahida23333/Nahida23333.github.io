【妇产学】妊娠滋养细胞疾病

发表于2025-09-29|更新于2025-09-29|妇产学

葡萄胎

葡萄胎因妊娠后胎盘绒毛滋养细胞增生、间质水肿，而形成大小不一的水泡，水泡间借蒂相连成串，形似葡萄而得名，又称水泡状胎块（hydatidiform mole）。

相关因素

正常妊娠 完全性葡萄胎 部分性葡萄胎
染色体核型 二倍体 二倍体 三倍体
染色体构成 1父+1母 2父 2父+1母
滋养细胞 正常 增生 增生
胎儿及附属物 有 无 异常

完全性葡萄胎

完全性葡萄胎的染色体核型为二倍体，均来自父系。其中90%为46,XX，系由1个细胞核缺如或失活的空卵（enucleate egg）与1个单倍体精子(23,X)受精，经自身复制为二倍体(46,XX)；另外，有10%核型为46,XY，系由1个空卵被2个单倍体精子(23,X)和(23,Y)同时受精而成。

部分性葡萄胎

分性葡萄胎的染色体核型90%以上为三倍体，合并存在的胎儿也为三倍体。最常见的核型是69,XXY，其余为69,XXX或69,XYY系由1个看似正常的单倍体卵子和2个单倍体精子受精或1个减数分裂缺陷的双倍体精子受精而成，所以1条多余的染色体也来自父方。

⭐临床表现

【复习】HCG的作用：①刺激卵泡内膜细胞，使月经黄体转换为妊娠黄体；②促进雄激素转化为雌激素，刺激孕酮的形成；③抑制母胎免疫；④与母体甲状腺细胞TSH受体结合，刺激甲状腺活性。
HCG检测+早孕表现。

  1. 卵巢黄素化囊肿，卵巢增大；
  2. 妊娠呕吐发生早、症状重，子痫前期征象；
  3. 绒毛侵袭性增加，存在恶变可能；
  4. 甲状腺功能亢进表现。
  5. 停经
  6. 阴道流血
  7. 子宫异常增大
  8. 腹痛

⭐辅助检查

超声检查、血HCG、X线胸片、血型血常规、肝肾功能、传染病检查。

  • 血HCG：约45%的完全性葡萄胎患者血清hCG水平在10万IU/L以上，最高可达240万IU/L。但也有少数葡萄胎，尤其部分性葡萄胎hCG升高不明显。

    辅助检查原则：先无创再有创、先便宜再昂贵、生命安全第一位原则。

确诊依赖于清宫后病例诊断。完全性葡萄胎免疫组化染色P57阳性。

⭐鉴别诊断

鉴别要点 完全性葡萄胎 部分性葡萄胎
大体检查 弥漫性绒毛水肿；无胚胎成分 部分绒毛水肿；可能存在胎儿组织
绒毛 弥漫性增大 由增大的绒毛和小的纤维化绒毛组成的两种绒毛
滋养细胞增生 显著增生 轻度增生
细胞异型性 可能较显著 轻度异型性
绒毛间质 水肿明显；血管消失；有核红细胞不可见 血管存在；有核红细胞可见
P57 免疫组化 细胞滋养细胞和绒毛间质细胞核染色阴性 细胞滋养细胞和绒毛间质细胞核染色阳性
DNA 核型 双雄源性二倍体 双雄单雌三倍体
  • 正常妊娠、水肿性流产：染色体核型分析；
  • 母源性三倍体：细胞来源鉴定。
  • 先兆流产、异位妊娠

⭐治疗

  • 清宫术：既是治疗手段，又是明确诊断的方法。

⭐随访

  1. 清宫后每周检测一次hCG，连续三次阴性，之后每月1次共6个月；
  2. 询问病史：月经情况，有无异常阴道流血，有无咳嗽、咯血及其他转移症状；
  3. 妇科检查：必要时做B超，X线胸片或CT检查等；
  4. 避孕：可靠避孕6个月。

妊娠滋养细胞肿瘤

包括侵袭性葡萄胎和绒癌。

临床表现

  1. 子宫及阴道：不规则阴道出血甚至危及生命的大出血；
  2. 肺转移：咳嗽、咯血、胸闷气短、胸痛；
  3. 肝脏转移：右上腹疼痛，破裂时发生腹腔内出血；
  4. 脑转移：头痛、抽搐、视物模糊等。
  5. 其他部位转移相关的症状：脾、膀胱、消化道、骨等。

辅助检查

  1. hCG检测
  2. 病理检查：有组织学结果时以病理组织学结果为准。
  3. 影像学检查

诊断

GTN患者很少有组织学的诊断，根据典型的临床表现和辅助检查结果等即可诊断。

滋养细胞肿瘤解剖学分期（FIGO，2000 年）

分期 描述
Ⅰ期 病变局限于子宫
Ⅱ期 病变扩散，但仍局限于生殖器（附件、阴道、子宫阔韧带）
Ⅲ期 病变转移至肺，有或无生殖系统病变
Ⅳ期 所有其他转移

滋养细胞肿瘤 FIGO/WHO 预后评分系统（2000 年）

治疗

化疗为主，手术和放疗为辅。

随访

  1. 复发通常发生在1年以内，没有关于随访的最佳计划推荐，需要终生随访；
  2. 检测hCG：第一年每月一次，第2—3年每三月一次，第4—5年每年一次；
  3. 随访内容同葡萄胎。
  4. 随访期间严格避孕，一般化疗停止一年后方可妊娠；
  5. 将来的生育、妊娠和后代不受影响。

预后

低危GTN治愈率100%，高危GTN预后较差。
