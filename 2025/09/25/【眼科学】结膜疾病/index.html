【眼科学】结膜疾病

发表于2025-09-25|更新于2025-09-28|眼科学
|浏览量:

大纲要求

教学内容

结膜的组成和检查方法、结膜炎的病因、临床表现、类型、诊断及治疗原则。

重点

细菌性、病毒性与过敏性结膜炎的鉴别诊断。

难点

结膜充血和睫状充血的鉴别。

结膜炎的病因及分类

根据结膜的病因可分为超急性、急性（＜3周）或亚急性、慢性结膜炎（＞3周）。

根据病因可分为感染性（细菌性、病毒性、衣原体性）和非感染性结膜炎。

结膜炎的临床症状

双眼同时或先后发病，急性或亚急性发病，可有传染流行史。其体征包括：

  1. 结膜充血：是结膜炎最常见的体征。
  2. 结膜分泌物增加：细菌性结膜炎多为脓性分泌物，病毒性结膜炎多为水样分泌物。
  3. 膜形成：分为假膜和真膜。前者多见于腺病毒结膜炎，去除后上皮依旧保持完整；后者常见于白喉棒状杆菌结膜炎，β-溶血性链球菌结膜炎，强行剥除后创面粗糙、易出血。
  4. 滤泡及乳头增生。

诊断

结膜炎的诊断主要依靠其临床症状和体征，必要时行结膜分泌物涂片、培养和细胞学检查。

  • 细菌性结膜炎：大量多形核白细胞和细菌；
  • 病毒性：可见大量单核细胞；
  • 衣原体性：可见大量中性粒细胞和淋巴细胞。

治疗

  1. 局部抗感染治疗：抗感染滴眼液滴眼，夜间眼膏涂眼。
  2. 必要时全身治疗：淋球菌感染者需及时静脉足量抗生素治疗。

预防

  1. 勤洗手、洗脸，不用手擦眼；
  2. 防接触传播，患者用具需隔离并消毒；
  3. 防交叉感染，医务人员检查病人后需洗手消毒；
  4. 防病毒暴发，做好公共场所的卫生管理工作。

急性细菌性结膜炎

病原体

表皮葡萄球菌、金黄色葡萄球菌

特点

  1. 潜伏期短1—3天，传染性强；
  2. 结膜充血水肿，分泌物为脓性；
  3. 一般不累及角膜。
  4. 白喉杆菌引起的急性膜性结膜炎或急性假膜性结膜炎。

超急性细菌性结膜炎

病原体

淋球菌、脑膜炎球菌

特点

  1. 潜伏期短（10小时至2—3天），病情进展迅速；
  2. 结膜炎症重，伴有大量脓性分泌物；
  3. 迅速引起角膜浑浊、浸润，周边或中央角膜溃疡，若治疗不及时，几天后可发生角膜穿孔，严重威胁视力。

新生儿淋球菌性结膜炎

  1. 多为产道感染，出生后7天发病者为产后感染；
  2. 脓漏眼： 新生儿淋球菌性结膜炎分泌物由病初的浆液性很快转变为脓性，脓液量多，不断从睑裂流出，故又有脓漏眼之称。

流行性角结膜炎

病原体

腺病毒8、19、29、37型腺病毒（人腺病毒D亚组）

特点

  1. 具有强传染性，通过接触传播；
  2. 起病急、症状重，双眼发病；
  3. 主要症状有眼红、疼痛、畏光、伴有水样分泌物。假膜形成、滤泡增生；
  4. 耳前淋巴结肿大和压痛；
  5. 2周后发展为局部的上皮下浸润，主要散步于中央角膜。

流行性出血性结膜炎

病原体

肠道病毒70型（偶为柯萨奇病毒A24型）

特点

  1. 潜伏期短，为18—48小时，可暴发流行；
  2. 分泌物为水样；
  3. 结膜下出血；
  4. 滤泡增生明显；
  5. 伴有上皮性结膜炎和耳前淋巴结肿大；、
  6. 可有发热、肌肉痛等全身症状。

过敏性结膜炎

常见致敏原

花粉、尘螨、霉菌、动物毛皮屑等。

分类

  1. 季节性过敏性结膜炎（SAC）：Ⅰ型超敏反应，好发于某个季节，其中眼痒是患者最主要的主诉。多数致敏原为花粉。
  2. 常年性过敏性结膜炎（PAC）：Ⅰ型超敏反应，本病致敏原以尘螨为主。
  3. 春季角膜结膜炎（VKC）：Ⅰ型、Ⅳ型超敏反应，睑结膜乳头增生或角膜缘胶冻样增生。
  4. 巨乳头状结膜炎（GPC）：以直径＞1mm的结膜乳头为主要临床特征，患者常有角膜接触镜、眼部假体或结膜缝线等诱因。
  5. 特应性角膜结膜炎（AKC）：除过敏性结膜炎的表现外，最主要的体征是面部伴发特应性皮炎。

临床表现

  1. 症状：双侧眼痒、眼红、眼肿、异物感及结膜囊分泌物增多。
  2. 体征：上睑结膜铺路样乳头，角膜胶原样增生，角膜盾型无菌性溃疡。

诊断

需要同时满足以下两项必要条件：

  1. 症状：眼痒，可伴有异物感，结膜囊分泌物增多；
  2. 体征：结膜充血、结膜乳头、角膜特异性病变特征至少一项。

治疗

  1. 脱离过敏原及健康教育
  2. 局部滴眼液治疗：包括抗组胺药、肥大细胞稳定剂、糖皮质激素、免疫抑制剂、人工泪液等。
