【眼科学】眼外伤

发表于2025-10-28|更新于2025-10-28|眼科学
|浏览量:

概述

分类

机械性眼外伤

按标准化分类可分为：

  • 闭合伤：眼球钝挫伤、板层裂伤、表浅异物
  • 开放性：裂伤（穿通、贯通伤、眼内异物）、眼球破裂
    按临床通用分类：眼球钝挫伤、眼球穿通伤、眼异物伤

非机械性眼外伤

化学伤、热烧伤、辐射伤

诊断和治疗

原则：救命→救眼→提高视力→改善外观

评估：

  1. 首先排除合并重要脏器损伤；
  2. 详细了解受伤发生过程；
  3. 全面评估眼部个组织情况；
  4. 院外评估及指导自救。

处理：

  1. 去除治病因素；
  2. 闭合伤口；
  3. 预防继发感染；
  4. 处理其他伤情；
  5. 判断预后。

眼球钝挫伤

常见原因

机械性钝力，如砖石、拳头、球类、跌撞、车祸、爆炸的冲击波。

角膜挫伤

虹膜睫状体挫伤

虹膜与瞳孔异常

  • 虹膜瞳孔缘撕裂及瞳孔括约肌断裂；
  • 虹膜根部离断——半月形缺损，瞳孔“D”字形；
  • 外伤性瞳孔扩大；
  • 睫状肌或支配神经受损。

前房积血

  • 红魔血管破裂导致
  • 根据积血占前房的容量分为三级：少于1/3为Ⅰ级；介于1/3—2/3为Ⅱ级；大于2/3为Ⅲ级。
  • 可导致继发性青光眼和角膜血染。

外伤性青光眼

晶状体挫伤

晶状体脱位或半脱位

  • 悬韧带全部或部分断裂
  • 视力下降
  • 可继发继发性青光眼

玻璃体积血

睫状体、视网膜或脉络膜的血管损伤引起。

脉络膜和视网膜损伤

脉络膜破裂

视网膜震荡和损伤

  • 视网膜震荡
  • 视网膜挫伤

外伤性黄斑裂孔

视神经挫伤

  • 可瞬间致不可逆盲
  • 晚期视神经萎缩，预后差

隐匿性眼球破裂

  • 视力严重下降
  • 低眼压
  • 结膜下血肿
  • 结膜透见色素
  • 前房加深
  • 瞳孔移位
  • 大量眼内出血

急救

  • 遵循眼外伤急救原则
  • 排除眼球破裂
  • 防治远期并发症

眼球穿通伤

原因

锐器的刺入、切割。创口规则，眼内组织损伤、嵌顿或脱出。

临床表现

角膜穿通伤

  • 单纯性：角膜伤口较小且规则，常自行愈合，无虹膜嵌顿。
  • 复杂性：伤口大，不规则，常有虹膜脱出集嵌顿，前房变浅。

角巩膜穿通伤

巩膜穿通伤

并发症

  • 感染性眼内炎：细菌和真菌的感染；（注意：抗菌药物不同，且真菌感染禁用糖皮质激素治疗）
    • 革兰阳性菌＞革兰阴性菌＞真菌（少见）
    • 充分散瞳、局部和全身应用大剂量抗生素和糖皮质激素
  • 外伤性增殖性玻璃体视网膜病变：过度的修复反应，纤维组织牵拉引起视网膜脱离
  • 交感性眼炎：外伤造成言捏抗原暴露并激发自身免疫应答所致。

急诊处理

  • 排除危及生命的重要脏器损伤
  • 急诊缝合伤口
  • 排除眼内异物
  • 预防感染

眼内异物

  • 眼穿通伤病史
  • 临床表现：异物的入口、通道、穿通伤的表现
  • 影像学检查：X线、CT、B超、MRI（非磁性！）

眼烧伤

酸碱化学伤

发病机制

酸烧伤

  1. 强酸使蛋白凝固坏死；
  2. 蛋白凝固组织酸向深层渗透。

碱烧伤

  1. 碱溶解脂肪和蛋白质；
  2. 迅速渗透到组织深层和眼内。

临床表现

轻度

  1. 眼睑与结膜轻度充血水肿；
  2. 不留疤痕，无明显并发症，视力多不受影响。

中度

  1. 眼睑皮肤水疱或糜烂；
  2. 结膜水肿，出现小片缺血坏死；
  3. 角膜明显充血水肿，上皮层完全脱落或形成白色凝固层；
  4. 治愈后遗留角膜斑翳，影响视力。

重度

  1. 结膜广泛缺血坏死，呈灰白色浑浊；角膜全层灰白或者呈瓷白色；
  2. 角膜溃疡或穿孔；
  3. 葡萄膜炎、继发性青光眼、白内障；
  4. 角膜白斑，前粘性角膜白斑、角膜葡萄肿或眼球萎缩。

急救和治疗

  1. 急救
  2. 后续治疗：早期治疗（抗生素、糖皮质激素）、晚期治疗
