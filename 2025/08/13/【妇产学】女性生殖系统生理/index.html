【妇产学】女性生殖系统生理

发表于2025-08-13|更新于2025-08-31|妇产学
|浏览量:

主要教学内容

教学基本要求

  1. 掌握月经的临床表现。
  2. 掌握卵巢的功能及其周期性变化。
  3. 掌握下丘脑、垂体、卵巢的生理功能及其内分泌调节轴。
  4. 熟悉月经周期女性生殖器的周期性变化。
  5. 了解女性一生各个阶段的生理特点。
  6. 了解其他内分泌腺功能对月经周期的影响。

重点与难点

重点

  1. 月经的临床表现。
  2. 卵巢的功能及周期性变化。
  3. 月经周期的神经内分泌调节。

难点

月经周期的神经内分泌调节。

知识点整理

女性生命各阶段生理变化

  1. 女性一生根据其年龄和生理特点可分为7个阶段：胎儿期（受精—胎儿娩出）、新生儿期（出生后4周）、 儿童期（出生4周—12岁） 、 青春期（10—19岁，发动通常始于8—10岁） 、 性成熟期/生育期（18岁左右开始，历时近30年） 、绝经过渡期（开始出现绝经趋势直至最后一次月经的时期）、 绝经后期（绝经后的生命时期）。
  2. 儿童早期（8岁以前）下丘脑—垂体—性腺轴功能处于抑制状态，卵泡无雌激素分泌，生殖器呈幼稚型。儿童后期，下丘脑促性腺激素释放激素（GnRH）抑制状态解除，卵巢中的卵泡有一定发育并分泌性激素，但仍达不到成熟阶段；乳房开始发育，女性特征开始显现。
  3. 青春期按照顺序先后经历 乳房萌发 （最初特征）、 肾上腺功能初现 （阴毛腋毛）、 生长加速 （月经初潮后开始减缓）、 月经初潮 （青春期重要标志）这4个不同的阶段，各阶段互有重叠。
  4. 女性青春期第一性征的变化是在促性腺激素的作用下，卵巢增大，卵泡开始发育和分泌雌激素，雌激素促进生殖器从幼稚型变为成人型。除生殖器外，女性其他特有的性征即第二性征，包括音调变高、乳房发育、阴毛及腋毛分布、骨盆横径发育大于前后径，以及胸、肩部皮下脂肪增多等，这些变化呈现出女性特征。
  5. 性成熟期是卵巢生殖机能与内分泌机能最旺盛的时期。此时卵巢功能成熟，出现周期性排卵；生殖器官各部及乳房，再卵巢分泌的性激素的作用下发生周期性变化。
  6. 性成熟的标志：①卵巢功能成熟、周期性排卵；②规律性月经。
  7. 绝经：绝经过渡期卵巢功能逐渐衰退，卵泡数量明显减少且易发生卵泡发育不全，因而月经不规律，常为无排卵性月经。最终由于卵巢内卵泡自然耗竭或剩余的卵泡对垂体促性腺激素丧失反应，导致卵巢功能衰竭，月经永久性停止，称为绝经。

卵巢功能及卵巢的周期性变化

  1. 卵巢为女性的性腺，其主要功能为 产生卵子并排卵 和 分泌女性激素 ，分别称为卵巢的生殖功能和内分泌功能。
  2. 卵巢周期：从青春期开始到绝经前，卵巢在形态和功能上发生周期性变化称为卵巢周期。（卵泡期→排卵→黄体期，平均28天）
  3. 卵泡期：自月经第1日至卵泡发育成熟称为卵泡期，需15天左右，此期由卵泡颗粒细胞和卵泡膜细胞分泌 雌激素 。排卵发生时的卵细胞已经完成第一次减数分裂。
  4. 排卵：卵细胞及其周围的“卵冠丘复合体”（包括次级卵母细胞、第一极体、透明带、放射冠）一起被排出的过程，多发生在下次月经来潮前14天左右。
  5. 黄体期：排卵日至下次月经来潮之间为黄体期，一般为14天。黄体于 排卵后7—8天达峰 ，9—10日开始退化。此期由黄体细胞分泌 雌激素和孕激素 。
  6. 雌激素的分泌呈现出两个高峰：①月经第7日卵泡分泌雌激素量迅速增加，于 排卵前 达高峰；②排卵后1—2日，黄体开始分泌雌激素使循环中雌激素含量又逐渐上升，在 排卵后7—8日黄体成熟 时，循环中雌激素形成又一高峰（低于第一高峰）。
  7. 孕激素的分泌特点：①卵泡期卵泡不分泌孕酮；②排卵前开始分泌少量孕酮；③排卵后黄体分泌孕酮逐渐增加，至 排卵后7—8天 黄体成熟时分泌量达最高峰，以后逐渐下降。（测定孕激素含量可以判断排卵是否发生）
  8. 卵巢周期呈现出较大的个体差异，主要为 卵泡中后期 的差异。月经期、黄体期时间差异不大。

例题：关于基础体温测定，哪项不正确：
A、双相体温提示有排卵
B、高温相持续3周以上，早孕的可能性大
C、基础体温曲线能反应黄体功能
D、基础体温测定对助孕有利
E、基础体温能反应胚胎生长情况
【正确答案】E

  1. 甾体激素的合成：①卵泡内膜细胞和颗粒细胞为排卵前雌激素的主要来源；②黄体细胞在排卵后分泌大量的孕激素和雌激素；③肾上腺和卵巢间质主要分泌雄激素（睾酮）。

  2. 雌激素合成的两细胞—两促性腺激素学说：卵巢雌激素的合成由卵泡膜细胞核颗粒细胞在FSH与LH的共同作用下完成。LH与卵泡膜细胞LH受体结合后可使胆固醇形成睾酮和雄烯二醇，后二者进入颗粒细胞内成为雌激素的前身物质；FSH与颗粒细胞上FSH受体结合后激活芳香化酶，将睾酮和雄烯二醇分别转化为雌二醇和雌酮，进入血液循环和卵泡液中。

  3. 卵巢分泌的其他类型激素；

    • 多肽激素
      • 抑制素包括抑制素A（$\alpha\beta_A$）、抑制素B（$\alpha\beta_B$），可选择性抑制垂体FSH的产生，增强LH活性；
      • 激活素包括激活素A（$\beta_A\beta_A$）、激活素AB（$\beta_A\beta_B$）和激活素B（$\beta_B\beta_B$），可以提高垂体对GnRH的反应，刺激垂体FSH的产生；
      • 卵泡抑制素：激活素与之结合后，失去刺激FSH产生的能力。
    • 细胞因子和生长因子：通过自分泌或旁分泌的形式，也参与卵泡生长发育的调节。

子宫内膜的周期性变化与月经

  1. 月经：指伴随卵巢周期性变化而出现的子宫内膜周期性脱落及出血。
  2. 月经周期：出血的第1日为月经周期的开始，两次月经第1日的间隔时间称为一个月经周期。根据组织学变化，可将月经周期分为增殖期、分泌期、月经期三个阶段。
  3. 月经期为 月经周期第1—4日 ，是黄体退化、 孕酮和雌激素撤退 的最终结果。此期子宫内膜功能层从基底层崩解脱离形成月经来潮。
  4. 增殖期为 月经周期第5—14日 ，与卵巢周期中的 卵泡期 相对应。可分为早、中、晚三期。早期（5—7日）腺上皮细胞呈立方体或低柱状，中期（8—10日）腺上皮细胞增生呈柱状、腺体数增多，晚期（11—14日）腺上皮细胞变为高柱状，增殖为假复层上皮。
  5. 分泌期为 月经周期第15—28日 ，与卵巢周期中的 黄体期 相对应。亦分为早、中、晚三期。早期（15—19日）腺上皮细胞开始出现含糖原的核下空泡；中期（20—23日）腺体内的分泌上皮细胞顶端胞膜破裂，细胞内的糖原溢入腺体，称为 顶浆分泌 。晚期（24—28日）子宫内膜呈海绵状，内膜腺体开口面向子宫腔，有糖原等分泌物溢出，间质更疏松水肿。
  6. 月经血特征：①暗红色，除血液外还有子宫内膜碎片、子宫颈黏液及脱落的阴道上皮细胞；②含有前列腺素及来自子宫内膜的大量纤维蛋白酶；③月经血不凝，出血量多或出血快可出现血凝块。
  7. 经期：每次月经持续时间称为经期， 一般为2—8日，平均4—6日 。
  8. 经量：一次月经的总失血量，正常月经量为 20—60ml ，超过80ml为月经过多。

月经周期的调节

  1. 下丘脑、垂体与卵巢之间相互调节、相互影响，形成一个完整而协调的神经—内分泌系统，称为 下丘脑—垂体—卵巢轴（HPO） 。
  2. 腺垂体的促性腺激素细胞分泌卵泡刺激素（FSH）和黄体生成素（LH）。
  3. FSH的主要生理作用：①直接促进窦前卵泡及窦状卵泡颗粒细胞增殖和分化，分泌卵泡液， 使卵泡生长发育 ；②激活颗粒细胞芳香化酶， 合成和分泌雌二醇 ；③在前一周期的黄体晚期及本次周期卵泡期早期，促使卵巢内窦状卵泡群的募集；④促使颗粒细胞合成分泌IGF及其受体、抑制素、激活素等物质，并与这些物质协同作用，调节优势卵泡的选择和非优势卵泡的闭锁退化；⑤在卵泡期晚期与雌激素协同， 诱导颗粒细胞形成LH受体 ，为排卵及黄素化做准备。
  4. LH的生理功能：① 在卵泡期刺激卵泡膜细胞合成雄激素 ，为雌二醇的合成提供底物；②排卵前 促使卵母细胞最终成熟并排卵 ；③在黄体期 维持黄体功能 ，促进孕激素、雌二醇和抑制素A的合成和分泌。

  7. 小剂量对下丘脑产生负反馈，大剂量产生正反馈的是 雌激素 。
  8. 避孕药中主要起避孕的物质是 孕激素 ，其机制为：①负反馈抑制垂体LH的分泌高峰，从而抑制排卵；②促使宫颈黏液粘稠，从而不利于精子穿行；③抑制子宫内膜，不利于受精卵着床；④影响输卵管功能，不利于精卵运输。除孕激素外还有一些雌激素，其作用为：①抑制排卵；②维持子宫内膜，避免突破性出血；③刺激肝脏合成性激素结合蛋白，后者可以与雄激素结合，从而起到降低雄激素的作用；④维持生理需要，如女性的生理特征。

其他内分泌腺及前列腺素对女性生殖系统的影响

  1. 甲状腺功能减退：青春期之前可使性发育障碍，使青春期延迟；生育期则出现月经异常，临床表现为月经过少、稀发甚至闭经。患者多合并不孕，自然流产、早产、胎儿畸形或神经认知功能缺陷发生率增加。
  2. 甲状腺功能亢进：子宫内膜过度增生，表现为月经过多、过频，甚至发生功能失调性子宫出血。进一步加重时可出现月经稀发、月经减少甚至闭经。
  3. 肾上腺皮质 是女性雄激素的主要来源。少量雄激素为正常女性的阴毛、腋毛、肌肉和全身发育所必需。过多雄激素可抑制下丘脑分泌GnRH并对抗雌激素，使卵巢功能受抑出现闭经，甚至男性化表现。
  4. 胰岛素依赖型糖尿病患者常伴有卵巢功能低下，胰岛素抵抗的高胰岛素血症患者中，过多胰岛素促进卵巢生成过多雄激素，从而引起高雄激素血症。

【流行病学】病因与病因推断
病因：病因是指那些能使人群发病概率增加的因素，当其中的一个或多个因素不存在时，人群中发生该种疾病的概率就下降。 必要病因：是指引起某种疾病发生所必须具备的条件，一旦该因素缺乏，疾病就不会发生；但又该因素的存在，却不一定会导致疾病的发生。 充分病因：一组必然导致疾病发生的最低限度的状态或条件。 病因模型三角模型该模型认为疾病的发生是宿主、环境、动因三要素共同作用的结果。正常情况下，三者通过相互作用保持动态平衡，人们呈健康状态。一旦三者中的一个要素发生变化，且超过该三角平衡所能维持的最高限度时，平衡被打破，人们将发生疾病。 轮状模型
