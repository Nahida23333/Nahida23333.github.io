【精神病学】精神分裂症及其他原发性精神病性疾病

精神分裂症

病因与发病机制

  1. 遗传
  2. 神经发育：新皮质形成时期神经细胞从大脑深部向皮质迁移过程中出现了紊乱；
  3. 神经影像：额叶功能亢进、额叶功能低下、内侧额叶皮质失活失败；
  4. 神经生化：多巴胺、谷氨酸、5—羟色胺、γ—氨基丁酸（GABA）；
  5. 心理社会因素。

临床表现

前驱期症状

  1. 情绪改变：抑郁、焦虑、情绪波动、易激惹等；
  2. 认知改变：出现一些古怪或异常的观念和想法等；
  3. 对自身和外界的感知改变；
  4. 行为改变：社交退缩或丧失兴趣、多疑敏感、职业功能水平下降；
  5. 躯体改变：睡眠和食欲改变、虚弱感、头痛、背痛、消化道症状；
  6. 少数青少年以突然出现的强迫症状为首发症状。

显症期症状

阳性症状

  1. 幻觉：幻听、幻视、幻嗅、幻味、幻触均可出现，但以幻听最常见。
  2. 妄想：在意识清醒的基础上持续出现某些离奇古怪或令人难以置信的妄想，常提示精神分裂症的可能。
  3. 瓦解症状群：思维形式障碍、思维过程障碍、奇异行为和紧张症行为、不适当的情感。

阴性症状

  1. 意志减退：患者从事有目的性活动的意愿和动机减退或丧失；
  2. 快感缺失：持续存在的不能从日常活动中发现和获得愉快感，尤其是对即将参与的活动缺乏期待快感；
  3. 情感迟钝：不能理解和识别别人的情感表露 和/或 不能正确表达自己的情感；
  4. 社交退缩：对社会关系的冷淡和对社交兴趣的减退或缺乏；
  5. 言语缺乏：言语的产生减少或缺乏。

焦虑、抑郁症状

激越症状

  1. 攻击暴力：激越、冲动控制能力减退、社交敏感度降低；
  2. 自杀：最可能原因（抑郁症状），促发因素（虚无妄想、命令性幻听、逃避精神痛苦、物质使用）

定向、记忆和智能

对时间、空间和人物一般能正确定向，意识通常清晰，一般记忆和智能无明显障碍。

自知力

常缺乏。

诊断要点

症状特点

一般来说，患者在意识清晰的基础上持续较长时间符合下述症状应想到精神分裂症可能：

  1. 持续妄想：夸大/关系/被害妄想；
  2. 持续幻觉：最常见的是幻听；
  3. 思维紊乱：如词不达意及思维松弛、语言不连贯、语词新作；
  4. 被动体验、被影响或被控制体验；
  5. 阴性症状：如情感平淡、思维缺乏或言语缺乏、意志缺乏、社交缺乏或兴趣丧失；
  6. 明显的行为紊乱：可以出现在任何有目的的活动中；
  7. 精神运动性症状：紧张症性不安或激越、作态、蜡样屈曲、违拗、缄默或木僵。

病程特点

多为持续性病程。

首次发作者通常要求在一个月及以上的大部分时间内，持续存在上述1—7中至少2项，且其中至少符合1—4中的一项。

其他特点

家族史、排除诊断等。

鉴别诊断

继发性精神病性障碍、其他精神病性障碍、心境障碍、焦虑症与强迫症、人格障碍。

继发性精神病性障碍

  1. 躯体疾病与精神症状的出现在时间上密切相关，病情的消长常与原发疾病相平行；
  2. 精神症状多在意识障碍的背景上出现，幻觉以幻视为主，较少有精神分裂症的“特征性”症状；
  3. 体格检查可有阳性发现；
  4. 实验室检查常可找到相关的证据。

其他精神病性障碍

  1. 分裂样精神障碍：病程不足一个月；
  2. 急性短暂性精神病性障碍：没有前驱症状的情况下突然起病，精神病性症状在2w内达峰。症状和性质波动大，通常在数天内完全缓解；
  3. 分裂情感障碍：一次疾病发作过程中，精神病性障碍和心境障碍（躁狂或抑郁）均很明显且差不多同时出现或消退；
  4. 妄想性障碍：妄想结构严密系统，内容有事实基础，思维有条理和逻辑，行为和情感与妄想内容一致，无智能和人格衰退，一般无幻觉或不为主要表现。

心境障碍

精神病性症状在心境症状改善时通常较快消失，不是疾病的主要临床相。

焦虑与强迫症

  1. 焦虑与强迫症多数由良好的自知力，了解自己的病情变化和处境，求治心切，情感反应强烈；
    精神分裂症早期虽可有自知力，但不迫切求治，情感反应亦不强烈。
  2. 仔细的病史询问及精神检查：可发现其他精神分裂症的症状；
  3. 一时难以诊断者：一定时间的随访观察，切忌轻易诊断。

人格障碍

  1. 人格障碍是一个固定的情绪、行为模式，一般无明显、持续的精神病性症状；症状表现是一个量的变化，无确切发病点；
  2. 精神分裂症病前病后有明显转折，情感与行为有质的异常，且具有某些中性精神病性症状。

病程与预后

病程

青少年起病，存在前驱期，呈恶化与缓解交替病程。

预后

  1. 异质性大；
  2. 近半数随访期间明显改善；
  3. 病程变化为疾病前5年最大，随后进入相对稳定的平台期；
  4. 总体预后差于分裂情感障碍和心境障碍；
  5. 病程和结局差异与选用的临床标准有关。

治疗

药物治疗（首选）

  1. 药物选择原则：根据依从性、疗效、耐受性、既往治疗体验、长期治疗计划、年龄、性别及经济状况综合考虑后选择；
  2. 药物使用原则：早期、适量、足疗程、单一用药、个体化用药；
  3. 药物治疗程序：
    • 急性治疗期：尽快控制症状，防止继发性伤害；
    • 巩固治疗期：防治疾病复燃，协助患者恢复病前社会功能；
    • 维持治疗期：防止疾病复发，进一步改善社会功能的整合和提高生活质量；
  4. 合并用药原则
    • 患者持续出现焦虑、抑郁和敌意等症状→相应药物对症处理；
    • 经合适的抗精神病药物治疗，但仍表现持续的阳性精神病性症状→辅助药物、电抽搐治疗（ECT）、经颅磁刺激治疗、联合应用不同种类的抗精神病药物、单独应用ECT。
  5. 安全监测与不良反应的处理。

物理治疗

电抽搐治疗（ECT）与改良电抽搐治疗（MECT）。

心理与社会干预

  1. 社会技能训练；
  2. 家庭干预：心理教育、家庭危机干预、以家庭为基础的行为治疗；
  3. 认知行为疗法（CBT）；
  4. 社区服务；
  5. 其他。

分裂情感障碍（SAD）

概述

在同一次疾病发作期间同时满足精神分裂症和心境障碍诊断要求的发作性疾病，精神分裂症症状和心境障碍症状可同时出现或相隔几天出现。

诊断要点

  1. 在疾病的同一次发作中，精神病性症状和情感性症状同时出现或只差几天出现；
  2. 该发作符合精神分裂症全部诊断要求的同时存在情感症状，且情感症状需满足以下心境发作（中重度抑郁发作、躁狂发作或混合发作）之一的诊断要求。

    要注意的是，抑郁发作的症状必须包括情绪低落，不能只依据兴趣减退或愉悦感降低。

  3. 精神病性症状和情感性症状共同存在至少一个月。
  4. 这种症状或行为不是其他躯体疾病所致，也不是物质或药物作用于中枢神经系统的效应所致。

预后

好于精神分裂症、差于精神分裂症。

分裂型障碍

以持久的（通常至少几年）表现为外表、语言和行为古怪而反常的模式，伴有感知和认知障碍、不寻常的信念，以及对人际关系感到不适病导致人际关系不良为特征。
