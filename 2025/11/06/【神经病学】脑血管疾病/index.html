【神经病学】脑血管疾病

发表于2025-11-06|更新于2025-11-07|神经病学
|浏览量:

脑血管疾病（CVD）是各种病因导致的脑循环障碍性疾病的总称，包括脑血管壁结构或管腔内血液血流异常等引发的局灶性或全面性神经功能缺损。

脑血管疾病的分类

缺血性脑血管病

  • 短暂性脑缺血发作
  • 缺血性卒中（脑梗死）：大动脉粥样硬化（LAA）性脑梗死、心源性栓塞性脑梗死、小动脉闭塞性脑梗死、其他原因所致脑梗死、原因未明的脑梗死
  • 脑动脉盗血综合征：锁骨下动脉盗血综合征、颈动脉盗血综合征、椎—基底动脉盗血综合征

出血性脑病

  • 蛛网膜下腔出血
  • 脑出血
  • 其他颅内出血

头颈部动脉粥样硬化、狭窄或闭塞（未导致脑梗死）

高血压脑病

原发性中枢神经血管炎

其他脑血管疾病

  • 脑底异常血管网病
  • 伴皮质下梗死和白质脑病的常染色体显性遗传病（CADASIL）和伴皮质下梗死和白质脑病的常染色体隐性遗传病（CADASIL）
  • 头颈部动脉夹层
  • 可逆性脑血管收缩综合征

颅内静脉系统血栓形成

无急性局灶性神经功能缺损的脑血管病

  • 无症状性脑梗死
  • 脑微出血

卒中后遗症

血管性认知障碍

  • 非痴呆性血管认知障碍
  • 血管性痴呆

短暂性脑缺血发作（TIA）

TIA是指脑或视网膜局灶性缺血所致的、未发生急性梗死的短暂性神经功能缺损发作；临床症状多在数分钟—数小时内恢复，不遗留神经功能缺损的症状和体征，且无急性脑梗死的影像学证据。

病因与发作机制

  1. 血流动力学改变：动脉严重狭窄基础上，血液急剧波动和下降导致一过性缺血。
  2. 微栓塞：微栓子阻塞小动脉导致缺血，栓子破裂或自发溶解时症状缓解。

临床表现

  1. 多发于50—70岁老年人，男多于女。
  2. 多伴有危险因素：高血压、糖尿病、血脂异常、动脉粥样硬化、心脏病等。
  3. 具有 突发性、发作性、短暂性、可逆性 的临床特征，而临床症状多种多样，取决于受累血管的分布（颈内动脉系统和椎—基底动脉系统）。
  4. 特殊临床表现：
    • 跌倒发作： 下肢突然失去张力而跌倒（脑干下部网状结构缺血）
    • 双眼视力障碍发作： 暂时性皮质盲（双侧大脑后动脉距状沟支缺血导致枕叶是皮质受累）
    • 短暂性全面性遗忘（TGA）： 一种突然起病的一过性记忆丧失，伴时间、空间定向力障碍，无意识障碍，患者的自知力存在，较复杂的皮质高级活动保留完整，无其他的神经系统异常表现，症状持续时间短，恢复后遗留有完全或部分对发作期事件的遗忘，预后良好。

辅助检查

  1. 常规检查：常见危险因素检查。
  2. 心电图及超声心电图：心源性栓子
  3. 颅脑CT和MRI：判断TIA或急性脑梗死。
  4. 颅内外血管病变检查：CTA、MRA（无创性血管成像技术），DSA（“金标准”，但有创）

诊断与鉴别诊断

诊断

  1. 诊断步骤：是否TIA→哪个系统的TIA→病因发病机制→危险因素评估
  2. 危险分层：ABCD2评分（年龄A、血压B、临床症状C、症状持续时间D、糖尿病D），0—3分为低危人群，4—5分为中危人群，6—7分为高危内容。

鉴别诊断

部分性癫痫发作、良性发作性位置性眩晕（BPPV）、偏头痛、其他

治疗

药物治疗

  1. 抗血小板聚集药物：阿司匹林、氯吡格雷。
  2. 抗凝治疗：伴有瓣膜性心房颤动（含阵发性）、风湿性二尖瓣病变及人工机械瓣膜等的TIA患者（感染性心内膜炎除外），口服华法林
  3. 其他

病因治疗

针对可能存在脑血管病危险因素进行积极有效干预。

手术和介入治疗

劲动脉内膜切除术（CEA）、经皮腔内血管成形术（PTA）。

预后

TIA患者发生急性缺血性脑卒中的风险明显高于一般人群。

脑梗死（缺血性脑卒中）

各种脑血管病变所致的脑部血液供应障碍，导致局部脑组织缺血、缺氧性坏死，而迅速出现相应神经功能缺损的一类临床综合征。

大动脉粥样硬化性脑梗死

病理生理

缺血型级联反应

  • 神经细胞在完全缺血、缺氧十几秒后即出现电位变化；
  • 20—30秒后大脑皮质的生物电活动消失；
  • 30—90秒后小脑和延髓的生物电活动也消失；
  • 脑血流中断5分钟，神经细胞发生不可逆损害，发生脑梗死。

治疗时间窗

急性脑梗死病灶由梗死核心区及其周围的缺血半暗带组成。有效挽救缺血半暗带组织的治疗时间，称为治疗时间窗（TTW）。

临床表现

一般特点

中老年多见，多有危险因素，部分在发病前有TIA发作。

不同脑血管闭塞的临床特点

  1. 颈内动脉闭塞的表现：
    • 单眼一过性黑矇，偶见永久性失明
    • Horner综合征
  2. 大脑中动脉闭塞的表现
    • 主干闭塞
      • 三偏症，即病灶对侧偏瘫（包括中枢性面舌瘫和肢体瘫痪）、偏身感觉障碍即偏盲；
      • 双眼向病灶侧凝视，优势半球受累出现失语，非优势半球受累出现体象障碍。
    • 皮质支闭塞
      • 上部分支闭塞：病灶对侧面部、上下肢瘫痪和感觉缺失，伴Broca失语（优势半球）和体象障碍（非优势半球）
      • 下部分支闭塞：对侧同向性上四分之一视野缺损，伴Wernicke失语（优势半球）、急性意识模糊状态（非优势半球）
    • 深穿支闭塞：对侧中枢性均等轻偏瘫、对侧偏身感觉障碍，可伴对侧同向性偏盲，皮质下失语（优势半球）。
  3. 大脑前动脉闭塞
    • 分出前交通动脉前的主干闭塞
      • 可因对策动脉侧支循环代偿而不出现症状。
      • 双侧动脉起源于同一个主干：双下肢截瘫，尿便失禁、意志缺失、运动性失语、额叶人格改变（双侧大脑半球的前内侧梗死）
    • 分出前交通动脉后的大脑前动脉远端闭塞
      • 对侧的足和下肢的感觉运动障碍，而上肢和肩部瘫痪轻，面部和手部不受累。
    • 皮质支阻塞
      • 对侧中枢性下肢瘫，可伴感觉障碍；
      • 对侧肢体短暂性共济失调，出现强握反射及精神症状。
    • 深穿支阻塞
      • 对侧中枢性面舌瘫、上肢近端轻瘫
  4. 大脑后动脉闭塞的表现
    • 单侧皮质支闭塞：对侧同向性偏盲
    • 双侧皮质支闭塞：完全性皮质盲
    • 大脑后动脉起始段的脚间支闭塞：中脑中央梗死和下丘脑综合征
      • Weber综合征：同侧动眼神经麻痹和对侧偏瘫
      • Claude综合征：同侧动眼神经麻痹和对侧共济失调
      • Benedikt综合征：同侧动眼神经麻痹和对侧不自主运动、震颤
    • 大脑后动脉深穿支闭塞
      • 丘脑穿通动脉闭塞：红核丘脑综合征（病灶侧舞蹈样不自主运动、意向性震颤、小脑性共济失调，和对侧偏身感觉障碍）。
      • 丘脑膝状体动脉闭塞：丘脑综合征（对侧神感觉障碍、自发性疼痛、感觉过度、轻偏瘫、共济失调、手部痉挛、舞蹈症、手足徐动症）
  5. 椎—基底动脉闭塞的表现（具体临床表现）
    • 闭锁综合征（locked—in syndrome）：基底动脉的脑桥支闭塞
    • 脑桥腹外侧部综合征（Millard—Gubler syndrome）：基底动脉短旋支阻塞
    • 脑桥腹内侧综合征/福维尔综合症（Foville syndrome）：基底动脉的旁中央支闭塞
    • 基底动脉尖综合征（top of the basilar syndrome）：基地动脉尖端闭塞，导致眼球运动障碍及瞳孔异常、觉醒和行为障碍。
    • 延髓背外侧综合征（Wallenberg syndrome）：小脑下后动脉或椎动脉供应延髓外侧的分支动脉闭塞。

特殊类型的脑梗死

  1. 大面积脑梗死
  2. 脑分水岭梗死（CWI）
  3. 出血性脑梗死
  4. 多发性脑梗死

辅助检查

溶栓指征的紧急筛查

血糖化验、全血细胞计数、凝血酶原时间（PT）、国际标准化比值（INR）、活化部分凝血活酶时间（APTT）。

卒中常规检查

颅脑CT（首选）、多模式CT、MRI。

血管病变检查

颈动脉超声、经颅多普勒超声（TCD）、磁共振血管成像（MRA）、高分辨磁共振成像（HRMRI）、CT血管成像（CTA）和数字减影血管造影（DSA）。

其他检查

动态心电图、超声心动图，蛋白C、蛋白S、抗凝血酶Ⅲ等。

鉴别诊断：脑梗死与脑出血的鉴别要点

鉴别点 脑梗死 脑出血
发病年龄 多为60岁以上 多为60岁以下
起病状态 安静或睡眠中 动态起病（活动中或情绪激动时）
起病速度 10余小时或1—2天症状达峰 10分钟至数小时症状达峰
全脑症状 轻或无 高颅压症状（头痛、呕吐、嗜睡、打哈欠）
意识障碍 无且较轻 多见且较重
神经体征 多为非均等性偏瘫 多为均等性偏瘫
CT检查 脑实质内低密度 脑实质内高密度
脑脊液 无色透明 可有血性

治疗

一般治疗

  1. 吸氧和通气支持
  2. 心脏监测和心脏病变处理
  3. 体温控制：抗感染、退热
  4. 血压控制
  5. 血糖

特异治疗

  1. 静脉溶栓：重组组织型纤溶酶原激活剂（rtPA）、尿激酶；时间窗为发病4.5小时内（rtPA）或发病6小时内（尿激酶溶栓）。
  2. 血管内介入治疗：血管内机械取栓、动脉溶栓、血管成形术。
  3. 抗血小板聚集治疗：阿司匹林（联合氯吡格雷）。
  4. 抗凝治疗
  5. 其他治疗：他汀类药物、降纤治疗、扩容治疗、神经保护治疗、高压氧和亚低温、中医中药治疗。

急性期并发症处理

  • 脑水肿和颅内压升高
    • 避免和处理引起高颅压的因素
    • 抬高头位（床头抬高＞30°）
    • 降颅压药物：甘露醇、高渗盐水、呋塞米
  • 梗死后出血转化：停用抗栓药物，再次启动抗栓治疗的时机根据情况权衡
  • 癫痫：抗癫痫治疗，再发癫痫按抗癫痫常规治疗
  • 深静脉血栓（DVT）和肺栓塞（PE）：尽早活动、抬高下肢；避免下肢（尤瘫痪侧）静脉输液
  • 压疮：定期反身、保持卫生；
  • 脑卒中后认知障碍和情感障碍

早期系统康复

早期开始二级预防

心源性脑栓塞

脑栓塞是指各种栓子随血流进入脑动脉，使血管急性闭塞或严重狭窄，导致局部脑组织缺血、缺氧性坏死，而迅速出现相应神经功能缺损的一组临床综合征。

心房颤动（房颤、AF）是心源性脑栓塞中最常见的原因。在心脏内壁和瓣膜形成的血栓或赘生物脱落后可阻塞脑动脉，引起脑栓塞。

80%见于颈内动脉系统，大脑中动脉尤为多见；20%见于椎基底动脉系统，其中基底动脉尖部和大脑后动脉多见。

小动脉闭塞性脑梗死

小动脉闭塞性梗死主要是指大脑半球或脑干深部的小穿支动脉，在高血压等各种疾病的基础上，血管壁发生病变（玻璃样变动脉硬化性病变及纤维素样坏死等），导致管腔闭塞，形成小的梗死灶。

脑出血（原发性非外伤性脑实质内出血）

病因与发病机制

病因

  • 高血压性脑细小动脉硬化（最常见）
  • 脑淀粉样病变、抗栓治疗、脑动静脉畸形、脑动脉瘤、烟雾病、血液病、肝脏疾病导致的出凝血功能障碍

发病机制

长期高血压→脑细小动脉发生玻璃样变性和纤维素样坏死，甚至形成微动脉瘤或夹层动脉瘤

血压骤然升高→血管破裂出血

临床表现

一般表现

  • 50y以上患者，男性稍多
  • 高血压病史，寒冷季节发病率高
  • 情绪激动或活动时发病
  • 症状在数分钟至数小时内达峰
  • 血压常明显升高

局灶性定位表现

基底节区出血

  1. 壳核出血（最常见）：豆纹动脉破裂所致
    • 病灶对侧偏瘫、偏身感觉障碍和同向性偏盲
    • 双眼向病灶侧凝视
  2. 丘脑出血：丘脑膝状体动脉或丘脑穿通动脉破裂
    • 对侧偏瘫、偏身感觉障碍
    • 感觉障碍重于运动障碍
    • 深感觉障碍更明显
    • 自发性疼痛和感觉过度。
  3. 尾状核头出血

脑叶出血

顶叶＞颞叶、枕叶、额叶

脑干出血

  1. 脑干出血
    • 小量出血：头痛、呕吐、交叉性瘫痪或偏瘫、眩晕、共济失调、两眼向病灶侧凝视麻痹或核间性眼肌麻痹。
    • 大量出血：意识障碍，出现双侧针尖样瞳孔、中枢性高热、呼吸节律紊乱、呕吐咖啡样胃内容物、眼球浮动、四肢瘫痪和去大脑强直。
  2. 中脑出血
    • 轻症：突然出现复视、眼睑下垂、一侧或双侧瞳孔扩大、眼球不同轴、眼震、同侧肢体共济失调。
    • 重症：意识障碍、四肢瘫痪、去大脑强直，可短期内死亡。
  3. 延髓出血：突然猝倒、意识障碍、血压下降、呼吸节律不规则、心律失常继而死亡。

小脑出血

小脑受损症状，如病灶侧共济失调、眼震和吟诗样语言等，多无瘫痪。

脑室出血

分原发性和继发性。

  1. 出血量少：头痛、呕吐、脑膜刺激征，无局灶性神经体征。
  2. 出血量多：梗阻性脑积水表现（昏迷、针尖样瞳孔、四肢瘫痪、去大脑强直发作、脑膜刺激征），丘脑下部受损（中枢性高热、大汗、消化道出血、血糖增高、尿崩症）

辅助检查

  1. CT平扫（首选）和CTA检查
  2. MRI和MRA检查
  3. DSA检查
  4. 脑脊液检查：如需排除颅内感染和蛛网膜下腔出血，可谨慎进行。
  5. 其他检查

诊断

  1. 中老年患者在活动中或情绪激动时突然发病；
  2. 迅速出现局灶性神经功能缺损症状和头痛、恶心、呕吐等高颅压表现，可伴有不同程度意识障碍；
  3. 颅脑CT出现脑出血征象；
  4. 排除其他疾病。

治疗

内科处理

  1. 一般处理：卧床休息、抬高床头，呼吸道通畅，镇静止痛剂，消化道出血——禁食，便秘——缓释剂；
  2. 脱水降颅压：渗透性脱水剂20%甘露醇；
  3. 调控血压：首先以脱水降颅压治疗为基础。
  4. 止血治疗：
    • 溶栓治疗相关脑出血：纤维蛋白原、PLT和含FⅧ的冷沉淀物；
    • 肝素治疗相关脑出血：鱼精蛋白；
    • 华法林相关脑出血：维生素K1.
    • 止血药物：6—氨基磷酸或氨甲苯酸。
  5. 亚低温治疗
  6. 并发症的防治：肺部感染、上消化道出血和水、电解质紊乱。

外科治疗

手术宜在早期（发病6—24小时内）进行。适应症：

  1. 中等量以上血肿：壳核出血≥30ml、丘脑出血≥15ml、小脑出血≥10ml，或血肿范围直径≥3cm。
  2. 合并梗阻性脑积水或脑疝。
  3. 重症脑室出血。
  4. 合并脑血管畸形、动脉瘤等需手术处理的血管病变。

康复治疗

预后

总体预后差，脑水肿、颅内压增高和脑疝形成的主要原因。

蛛网膜下腔出血（SAH）

蛛网膜下腔出血泛指各种病因导致脑血管破裂后，血液流入蛛网膜下腔引起的病理过程。

病因与发病机制

  1. 病因：颅内动脉瘤（最常见病因）、脑血管畸形（尤动静脉畸形）、其他病因（烟雾病、抗栓治疗、血液病、肝脏疾病）；
  2. 发病机制：动脉瘤血管壁破裂。

临床表现

一般临床表现

分级 标准
0 未破裂动脉瘤
Ⅰ 无症状或轻微头痛
Ⅱ 重度头痛、脑膜刺激征、脑神经麻痹
Ⅲ 嗜睡、意识混沌、轻度局灶性神经体征
Ⅳ 昏迷、中或重度偏瘫、有早期去大脑强直或自主神经功能紊乱
Ⅴ 昏迷、去大脑强直
  1. 头痛：最常见，为突发剧烈胀痛或爆裂样头痛，或称“雷击样疼痛”。
  2. 脑膜刺激征：以颈强直最多见。
  3. 神经缺损表现
    • 眼部表现：玻璃体下片状出血、眼球活动障碍
    • 其他表现：癫痫发作、偏瘫或失语等局灶性神经功能症状。

主要并发症

  1. 再出血：蛛网膜下腔出血的严重并发症，指病情稳定后再次发生剧烈头痛、呕吐、癫痫发作、昏迷甚至去大脑强直发作、颈强直等脑膜刺激征加重，复查影像学或腰穿证实再出血。
  2. 脑血管痉挛：3—4天出现，7—10天达峰，14—21天逐渐缓解。
  3. 脑积水：积性脑积水和慢性脑积水。
  4. 其他：癫痫、应激性消化性溃疡、神经源性心肌损伤、急性肺水肿、高血糖和低钠血症。

辅助检查

  1. 颅脑CT平扫（首选）和MRI
  2. 无创血管成像检查：CTA、MRA；
  3. 有创血管成像检查：DSA（“金标准”）
  4. 腰椎穿刺检查：临床疑似蛛网膜下腔出血而影像学检查结果为阴性。
  5. 其他：TCD、血常规、凝血功能、肝功能检查。

诊断

  1. 突发剧烈头痛、呕吐、脑膜刺激征(+)；
  2. 头颅影像学显示蛛网膜下腔出血征象。

  2. 颅内感染
  3. 脑膜癌病
  4. 其他疾病

治疗

治疗目的：防治再出血、降低颅内压、减轻脑损伤并挽救生命

一般处理

预防再出血

  1. 绝对卧床休息：4—6周；
  2. 病因治疗：动脉瘤破裂所致——血管内栓塞或开颅手术夹闭动脉瘤，动静脉畸形所致——血管内治疗、外科切除或立体定向反射治疗。
  3. 血管管理：保持收缩压＜160mmHg。
  4. 抗纤溶药物：氨基环酸或氨基己酸。

降低颅内压

渗透性脱水剂

防治脑血管痉挛

尼莫地平，维持正常循环血容量

治疗脑积水

脑脊液外引流术、反复行腰椎穿刺释放血性脑脊液。

治疗癫痫

一些题目

题目1

  1. 男，56岁的心房颤动患者，突然发生命名物名困难。二周来共发生过5次，每次持续2—15秒。查体无神经系统异常，脑CT无异常。（可能为TIA）主要累计的血管是（ D ）

A.基底动脉系
B.椎动脉系
C.颈内动脉系
D.大脑后动脉
E.大脑前动脉

颈内动脉系统：
大脑前动脉——人格情感障碍、运动性失语、对侧肢体无力；（“人前谈感情”）
大脑中动脉——三偏症
颈内动脉眼支——视物模糊、单眼黑矒失明

椎基底动脉系统： 眩晕、平衡障碍、眼球运动异常和复视
小脑后下动脉（椎动脉）——延髓背外侧综合征/Wallenburg综合征
大脑后动脉（基底动脉）——命名性失语（“人后忘哥名”）

题目2

最易发生脑血栓形成的动脉为（ B ）。

A.大脑前动脉
B.颈内动脉及大脑中动脉
C.基底动脉
D.大脑后动脉
E.椎—基底动脉

脑血栓：颈内动脉
脑栓塞：大脑中动脉
脑出血：基底核的桥核和内囊区
蛛网膜下腔出血：颈内动脉及其分叉处

题目3

女性，58岁，晨起后觉头晕，下次语言不清，右侧肢体运动不便，次日症状加重。CT左侧内囊低密度影。下列诊断正确的是（ A ）。

A.脑血栓形成
B.脑栓塞
C.蛛网膜下腔出血
D.脑出血
E.短暂性脑出血

脑血栓：中年女性好发，安静时起病，高血压、糖尿病、高血脂。
脑栓塞：青年人好发，激动时起病，房颤。

题目4

男，71岁，8小时前于日常活动中出现右侧肢体无力，且逐渐加重，查体：BP160/98mmHg，神志清，右鼻唇沟浅，右侧肢体肌力4级，右侧Babinski症阳性，右偏身感觉减退，头颅CT未见异常，诊断为脑血栓形成。对该患者最适宜的治疗措施是（ C ）。

A.溶栓治疗
B.积极降压
C.抗血小板聚集治疗
D.止血治疗
E.加强脱水

该患者发病时间已经超过溶栓治疗的时间窗，因此只能先抗血小板聚集。

题目5

高血压脑出血患者来院时昏迷，已脑疝，应首先采取的急救措施是（ D ）。

A.开颅手术
B.腰穿放脑脊液
C.脑室穿刺
D.静脉快速滴注甘露醇
E.静脉注射50%葡萄糖

已经发生脑疝，开颅手术来不及了，先打甘露醇脱水治疗，尽可能争取时间。

题目6

对蛛网膜下腔出血诊断具有提示作用的体征是（ E ）。

A.视乳头水肿
B.感觉障碍
C.脑神经麻痹
D.脑膜刺激征
E.玻璃体下出血

注意，脑出血破入脑室时亦可出现脑膜刺激征。因此，玻璃体下出血会比脑膜刺激征更具有代表性。
