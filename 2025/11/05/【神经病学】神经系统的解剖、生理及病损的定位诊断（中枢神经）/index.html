【神经病学】神经系统的解剖、生理及病损的定位诊断（中枢神经）

发表于2025-11-05|更新于2025-11-06|神经病学
|浏览量:

神经结构病损后出现的症状可分为4组：

  • 缺损症状：神经结构缺损时正常功能的减弱或消失；
  • 刺激症状：神经结构受激惹后引起的过度兴奋表现。
  • 释放症状：高级中枢受损后，原来受其抑制的低级中枢因抑制解除而出现功能亢进；
  • 断联休克症状：指CNS局部发生严重损害时，引起功能上和受损部位有密切联系的远隔部位神经功能短暂丧失。

大脑半球

额叶

解剖功能与生理功能

  1. 皮质运动区： 位于中央前回，支配对侧半身的随意运动。身体各部位代表区在此排列由上向下呈“倒人状”。
  2. 运动前区： 锥体外系皮质中枢；与共济运动有关；自主皮质神经中枢的一部分，肌张力抑制区。
  3. 皮质侧视中枢： 双眼同向侧视运动。
  4. 书写中枢： 优势半球额中回后部。
  5. 运动性语言中枢（Broca区）： 管理语言运动。
  6. 额叶前部： 广泛的联络纤维，与记忆、判断、抽象思维、情感和冲动行为有关。

病损表现与定位诊断

外侧面

  • 额极病变：精神障碍，表现为记忆力和注意力减退，表情淡漠，反应迟钝。
  • 中央前回病变：
    • 刺激性病变：对侧上下肢或面部的抽搐（Jackson抽搐），或继发性全身性癫痫发作；
    • 破坏性病变：单瘫。
  • 额上回后部病变：可产生对侧上肢强握反射和摸索反射。
  • 额中回后部病变：
    • 刺激性病变：双眼向病灶对侧凝视；
    • 破坏性病变：双眼向病灶侧凝视，更后部位的病变——书写不能。
  • 优势侧额下回后部病变：运动性失语（Braca失语）。

内侧面

  • 后部旁中央小叶：对侧膝以下瘫痪。
  • 矢状窦旁脑膜瘤：压迫双侧下肢运动区而使其瘫痪，伴有尿便障碍。

底面

  • 额叶眶面：饮食过量、胃肠蠕动过度，多尿、高热、出汗、皮肤血管扩张。
  • 额叶底面肿瘤：福-肯综合征（同侧嗅觉丧失和视神经萎缩，对侧视神经乳头水肿）。

顶叶

解剖结构和生理功能

  1. 皮质感觉区： 对侧肢体深浅感觉信息，代表区排列呈“倒人状”。
  2. 运用中枢： 复杂动作和劳动技巧。
  3. 视觉性语言中枢/解读中枢： 理解看到的文字和符号。

病损表现与定位诊断

中央后回和顶上小叶

  • 破坏性病变：病灶对侧肢体复合性感觉障碍。
  • 刺激性病变：病灶对侧肢体的部分性感觉性癫痫。

顶下小叶（缘上回和角回）

  • 体象障碍：顶叶病变。
  • 格斯特曼综合征：优势侧角回损害，表现为计算不能（失算症）、手指失认、左右辨别不能（左右失认症）、书写不能（失写症），有时伴失读。
  • 失用症：优势侧缘上回病变。

颞叶

解剖结构与生理功能

  1. 感觉性语言中枢（Wernicke区）
  2. 听觉中枢
  3. 嗅觉中枢
  4. 颞叶前部： 与记忆、联想、比较等高级神经活动。
  5. 颞叶内侧面： 海马与记忆、精神、行为和内脏功能有关。

病损表现与定位诊断

  • 优势半球颞上回后部（Wernicke区）：感觉性失语（Wernicke失语）（能听见说话声音而不能理解其含义）
  • 优势半球颞中回后部：命名性失语（能说出一个物品的用途而说不出其名称）
  • 颞叶钩回损害：“钩回发作”（幻嗅、幻味，舔唇、咀嚼）
  • 海马损害：癫痫，严重近记忆障碍。
  • 优势侧颞叶广泛病变或双侧颞叶病变：精神症状（人格、情绪、记忆）
  • 颞叶深部视辐射纤维和视束：双眼对侧视野的同向上象限盲。

枕叶

解剖结构和生理功能

围绕距状裂的皮质为视觉中枢，亦称纹状区，接受外侧膝状体传来的视网膜神经冲动。

病损表现与定位诊断

视觉中枢

  1. 双侧视觉中枢：皮质盲（全盲，视物不见但对光反射存在）
  2. 一侧视中枢病变：对侧视野同向性偏盲，而中心视野不受影响，称黄斑回避。
  3. 距状裂以下舌回：对侧同向性上象限盲。
  4. 距状裂以上楔回：对侧同向性下象限盲。

优势侧纹状区周围病变

视觉失认：患者并非失明，但对图形、面容或颜色等失去辨别能力，有时需借助触觉方可辨认。

顶枕颞交界区病变

视物变形：患者看物体变大、变小、形状歪斜及颜色改变等现象。

导叶、边缘叶

内囊

  1. 完全性内囊损害： “三偏”综合征（病灶对侧偏瘫、偏身感觉障碍、偏盲），多见于脑出血及脑梗死。
  2. 部分性内囊损害： 偏瘫、偏身感觉障碍、偏盲、偏身共济失调、一侧中枢性面舌瘫、运动性失语中的1个或多个症状。

基底神经节/基底核

解剖结构与生理功能

病损表现与定位诊断

  • 新纹状体：肌张力减低—运动过多综合征（舞蹈样动作、手足徐动症、偏侧投掷症）
  • 旧纹状体及黑质病变：肌张力增高—运动减少综合征（肌张力增高、动作减少、静止性震颤）

间脑

丘脑

解剖结构与生理功能

各种感觉（嗅觉除外）传导的皮质下中枢和中继站。

病损表现与定位诊断

  • 丘脑外侧核群尤其是腹后外侧核和腹后内侧核受损：对侧偏身感觉障碍，特点：
    • 各种感觉均发生障碍
    • 深感觉和精细触觉障碍＞浅感觉障碍
    • 肢体及躯干＞面部
    • 深感觉障碍所致共济失调
    • 感觉异常
    • 丘脑痛（对侧肢体自发性疼痛）
  • 丘脑至皮质下（锥体外系）诸神经核的纤维联系受损：面部表情分离性运动障碍
  • 丘脑外侧核群与红核、小脑、苍白球的联系纤维受损：对侧偏身不自主运动。
  • 丘脑前核与下丘脑及边缘系统的联系受损：情感障碍（情绪不稳强哭强笑）

下丘脑

解剖结构与生理功能

  1. 视前区（视前内侧核、视前外侧核）：体温调节；
  2. 视上区：水代谢（视上核）、糖代谢（室旁核）；
  3. 结节区：性功能（腹内侧核）、脂肪代谢（背内侧核）；
  4. 乳头体区（下丘脑后核、乳头体核）：产热保温。

病损表现与定位诊断

  • 视上核、室旁核及其纤维束：中枢性尿崩症；
  • 下丘脑散热和产热中枢：体温调节障碍；
  • 下丘脑饱食中枢（腹内侧核）和摄食中枢（灰结节外侧区）：摄食异常。
  • 下丘脑视前区与后区网状结构：睡眠觉醒障碍。
  • 下丘脑腹内侧核和结节区：生殖功能和性功能障碍；
  • 下丘脑后区和前区损害：自主神经功能障碍。

上丘脑、底丘脑

脑干

解剖结构与生理功能

  1. 脑干神经核
    • 中脑：第Ⅲ、Ⅳ对脑神经核团
    • 脑桥：第Ⅴ、Ⅵ、Ⅶ、Ⅷ对脑神经核团
    • 延髓：第Ⅸ、Ⅹ、Ⅺ、Ⅻ对脑神经核团
  2. 脑干传导束
  3. 脑干网状区域

病损表现与定位诊断

交叉性瘫痪：病灶侧脑神经周围性瘫痪和对侧肢体中枢性瘫痪及感觉障碍。

延髓

⭐延髓上段的背外侧区：瓦伦贝格综合征（Wallenberg syndrome）


常见于小脑下后动脉、椎—基底动脉或外侧延髓动脉缺血性损害。

  1. 眩晕、恶心、呕吐及眼震（前庭神经核）
  2. 病灶侧软腭、咽喉肌瘫痪（疑核及舌咽、迷走神经）
  3. 病灶侧共济失调（绳状体及脊髓小脑束、部分小脑半球）
  4. 病灶侧Horner综合征（交感神经下行纤维）
  5. 交叉性感觉障碍：同侧面部痛、温觉消失（三叉神经脊束核）
  6. 对侧偏身痛、温觉减退或丧失（脊髓丘脑侧束）

延髓中腹侧损害：德热里钠综合征

  1. 病灶侧舌肌瘫痪及肌肉萎缩（舌下神经）；
  2. 对侧肢体中枢性瘫痪（锥体束）；
  3. 对侧肢体触觉、位置觉、振动觉减退及丧失（内侧丘系）。

脑桥

脑桥腹外侧部损害：米亚尔—居布勒综合征

  1. 病灶侧眼球不能外展（展神经麻痹）及周围性面神经麻痹（面神经核损害）
  2. 对侧中枢性偏瘫（锥体束）
  3. 对侧偏身感觉障碍（内侧丘系和脊髓丘脑束损害）

脑桥腹内侧部损害：福维尔综合症

  1. 病灶侧眼球不能外展（展神经麻痹）及周围性面神经麻痹（面神经核损害）
  2. 双眼向病灶对侧凝视（脑桥侧视中枢及内侧纵束损害）
  3. 对侧中枢性偏瘫（锥体束损害）

脑桥背外侧部损害：雷蒙—塞斯唐综合征

与Wallenburg综合征相类似。

⭐双侧脑桥基底部病变：闭锁综合征（locked—in syndrome）

又称去传出状态，主要出现于基底动脉脑桥分支双侧闭塞。

  1. 意识清醒，语言理解无障碍
  2. 双侧中枢性瘫痪，只能以眼球上下运动示意，眼球水平运动障碍
  3. 不能讲话，双侧面瘫，构音及吞咽运动均障碍；
  4. 不能转颈耸肩，四肢全瘫，可有双侧病理反射（+）

中脑

⭐一侧中脑大脑脚脚底损害：韦伯综合症（Weber syndrome）

  1. 病灶侧除外直肌和上斜肌外的所有眼肌麻痹，瞳孔散大（动眼神经麻痹）
  2. 对侧中枢性面舌瘫和上下肢瘫痪（锥体束损害）

中脑被盖部腹内侧核损害：红核综合征/贝内迪克特综合征

  1. 病灶侧除外直肌和上斜肌外的所有眼肌麻痹，瞳孔散大（动眼神经麻痹）
  2. 对侧肢体震颤、强直（黑质损害）或舞蹈、手足徐动及共济失调（红核损害）
  3. 对侧肢体深感觉和精细触觉障碍（内侧丘系障碍）

小脑

  1. 主要维持躯体平衡，控制姿势和步态，调节肌张力和协调随意运动的准确性。
  2. 病变最主要的症状是共济失调。

脊髓

解剖结构与生理功能

功能主要有两方面：①上下行传导通路的中继站；②反射中枢。

病损表现与定位诊断

不完全性脊髓损害

  1. 前角损害：节段性下运动神经元性瘫痪，常伴有肌束震颤，肌电图上出现巨大综合电位。
  2. 后角损害：痛温觉丧失而触觉保留的分离性感觉障碍。
  3. 中央管附近的损害：双侧对称的分离性感觉障碍（痛温觉减弱或丧失而触觉保留）
  4. 侧角损害：$C_8—L_2$侧角（脊髓交感神经中枢，血管舒缩功能障碍、泌汗障碍、营养障碍），$C_8—T_1$（Horner综合征），$S_{2—4}$（副交感中枢，膀胱直肠功能障碍和性功能障碍）
  5. 前索损害：对侧病变水平以下粗触觉障碍。
  6. 后索损害：振动觉、位置觉障碍及感觉性共济失调。
  7. 侧索损害：病变水平以下同侧上神经元性瘫痪和对侧痛、温觉障碍。
  8. 脊髓束性损害：薄束和楔束（深感觉障碍）、锥体束（中枢性瘫痪）、脊髓小脑束（小脑共济失调）。
  9. 脊髓半侧损害：脊髓半切综合征，又称布朗—塞卡综合征
    • 病变节段以下同侧上运动神经元性瘫痪、深感觉障碍、精细触觉障碍、血管舒缩功能障碍，以及对侧痛、温觉障碍；
    • 对侧传导束性感觉障碍平面铰脊髓损害节段水平低。

脊髓横贯性损害

  1. 高颈椎（$C_{1—4}$）：损害平面以下各种感觉缺失，四肢呈上运动神经元瘫痪，括约肌障碍，四肢和躯干多无汗。
  2. 颈膨大（$C_5—T_2$）：双上肢呈下运动神经源性瘫痪，双下肢呈上运动神经源性瘫痪。病灶平面以下各种感觉缺失，可有肩部和上肢的放射性痛、尿便障碍等。
  3. 胸椎（$T_3—L_2$）：损害平面以下各种感觉缺失，双下肢呈上运动神经元性瘫痪。
    • 比弗症：病变位于$T_{10—11}$时可导致腹直肌下半部无力，当患者与仰卧位用力抬头时，可见脐孔被腹直肌上半部牵拉而向上移动。
  4. 腰膨大（$L_1—S_2$）：双下肢下神经元性瘫痪，双下肢及会阴部各种感觉缺失，括约肌障碍。
  5. 脊髓圆锥（$S_{3—5}$和尾节）：肛门周围和会阴部感觉丧失，呈鞍状分布，肛门反射消失和性功能障碍。
  6. 马尾神经根：与脊髓圆锥相似，但可为单侧或不对称。

一些题目

题目1

病变对侧偏身深、浅感觉障碍，伴自发性疼痛及感觉过敏，病变部位在（ C ）。

A.顶叶感觉皮质
B.内囊或基底节区
C.丘脑
D.中脑
E.脑桥

遇到偏身感觉障碍则考虑内囊和丘脑，存在“三偏”症为内囊损害，有自发性疼痛则为丘脑损害。

题目2

伸舌偏右，右侧舌肌萎缩伴肌束颤动，左上、下肢中枢性瘫，病变位于（ A ）。

A.右侧延髓
B.左侧延髓
C.双侧延髓
D.右侧脑桥及延髓
E.左侧脑桥及延髓

脑干损害表现为交叉性瘫痪，即患侧面瘫+健侧肢体瘫,据此我们可以定位为右侧损害。右侧舌肌萎缩说明舌下神经受损，舌下神经为第12对脑神经，舌下神经核位于延髓，故为右侧延髓损害。

题目3

男，40岁。渐起走路不稳2年，有共济失调障碍，步幅加宽，左右摇摆。最可能受损的部位是（ B ）。

A.小脑半球
B.小脑蚓部
C.丘脑
D.延髓
E.大脑半球

小脑蚓部——同侧躯干性共济失调：平衡障碍、醉汉步态、龙伯格征(+)；
小脑半球——同侧肢体性共济失调：指鼻试验(+)。
