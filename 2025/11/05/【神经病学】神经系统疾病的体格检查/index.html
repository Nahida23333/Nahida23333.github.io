【神经病学】神经系统疾病的体格检查

发表于2025-11-05|更新于2025-11-05|神经病学
|浏览量:

一般检查

一般状况检查

生命体征

体温

正常人体温平均37℃（口测法：36.3—37.2℃），24h内波动≤1℃。

脉搏

一般检查桡动脉。

心率

正常成人在安静、清醒状态下为60—100次/分，老年人偏慢、女性偏快、儿童较快。

呼吸

呼吸节律正常

深快的规律性呼吸（Kussmaul呼吸）：DKA、尿毒症、败血症；

浅快的规律性呼吸：休克、心肺疾病或安眠药中毒引起的呼吸衰竭。

呼吸节律异常

呼吸模式 损害水平 瞳孔 反射性眼球运动 疼痛反应
潮式呼吸 间脑 小，对光反射(+) 头眼反射存在 伸展过度
中枢神经源性过度呼吸 中脑被盖部 不规则，对光反射(±) 病变侧头眼反射消失 去皮质强直
长吸式呼吸 中脑下部和脑桥上部 针尖大小，对光反射(±) 病变侧头眼反射消失 去大脑强直
丛集式呼吸 脑桥下部 针尖大小，对光反射(±) 前庭眼动反射消失 去大脑强直
共济失调式呼吸 延髓上部 针尖大小，对光反射(±) 前庭眼动反射消失 弛缓或下肢屈曲
  1. 潮式呼吸/陈—施呼吸：呼吸由浅慢逐渐变为深快，再由深快变为浅慢，随后出现一段呼吸暂停，然后重复上述周期性呼吸。
  2. 中枢神经源性过度呼吸：呼吸深、均匀、持久，可达40—70次/分。
  3. 长吸式呼吸：吸2—3次呼1次或吸足气后呼吸暂停。
  4. 丛集式呼吸：频率、幅度变化不一的周期性呼吸。
  5. 共济失调式呼吸：呼吸频率和时间均不规律。

血压

体味或呼吸气味

酒味——饮酒或酒精中毒；烂苹果味——DKA；肝臭味——肝性脑病；氨味或尿味——尿毒症；大蒜味——敌敌畏中毒。

发育和体型、营养状态、面容表情、体位、语言语调和构音、姿势与步态、皮肤黏膜、头颈部、胸腹部、躯干与四肢

意识状态的检查

  1. 意识： 大脑功能活动的综合表现，是人对自身及外界环境进行认识和做出适宜反应的基础，包括觉醒状态和意识内容两个组成。
  2. 觉醒状态： 与睡眠呈周期性交替的清醒状态，由脑干网状激活系统和丘脑非特异性核团维持和激活。
  3. 意识内容： 人的知觉、思维、记忆、注意、智能、情感、意志、活动等心理活动（精神活动），还有通过言语、视觉、听觉、技巧性运动及复杂反应与外界环境保持联系的机敏力，属大脑皮质的功能。
  4. 意识障碍 可根据觉醒水平分为嗜睡、昏睡、昏迷，可根据意识内容改变程度分为意识模糊、谵妄状态，也包括去皮质综合征、去大脑强直、无动性缄默症、植物状态等特殊类型。

眼征

  1. 瞳孔：大小、形状、对称性、直接和间接对光反射
  2. 眼底：视神经乳头水肿、出血
  3. 眼球位置：眼球突出或凹陷
  4. 眼球运动

对疼痛刺激反应

  1. 去皮质强直：疼痛时上肢屈曲、下肢伸直，与丘脑或大脑半球病变有关；
  2. 去大脑强直：疼痛时四肢伸直、肌张力增高或角弓反张，提示中脑功能受损。

    去皮质强直和去大脑强直均不能精准定位。

瘫痪体征

坠落试验：

  1. 检查上肢时将患者双上肢同时托举后突然放开任其坠落，瘫痪侧上肢迅速坠落且沉重，无瘫痪侧则向外倾倒，缓慢坠落。
  2. 检查下肢时将患者一侧下肢膝部屈曲提高，足跟着床，突然松手时瘫痪肢体不能自动伸直，并向外倾倒，无瘫痪侧肢体则呈弹跳性伸直，并能保持足垂直位。

脑干反射

  1. 睫脊反射：给予颈部皮肤疼痛刺激引起双侧瞳孔散大→下位脑干、颈椎、上胸段脊髓及颈交感神经正常。
  2. 角膜反射：三叉神经与面神经，一侧消失提示同侧面神经（脑桥）病变，双侧消失提示中脑或脑桥受损。
  3. 头眼反射/玩偶眼试验：检查者轻扶患者头部，向左右、上下转动，患者眼球向与头部运动相反的方向移动，随后逐渐回到居中位置。（脑干病变时消失）
  4. 前庭眼动反射/冷热水试验：用注射器向一侧外耳道注入1ml冰水，脑干功能完整→双眼球缓慢转向灌注侧，同时发生朝向对侧的水平眼震。

脑膜刺激征

颈强直、Kernig征、Brudzinski征等。

意识障碍的其他体征

精神状态和高级皮质功能检查

高级皮质功能检查分为认知和非认知功能两大部分，前者包括记忆力、计算力、定向力，失语、失用、失认，抽象思维和判断、视空间等方面，后者包括人格改变、行为异常、精神症状和情绪改变等。

记忆

  1. 瞬时记忆检查方法：顺行性数字广度检测。
  2. 短时记忆检查方法
  3. 长时记忆检查方法

计算力

从100中连续减7。

定向力

时间定向力、地点定向力、人物定向力。

失语

  1. 口语表达：言语流畅性，有无语音障碍，有无语音障碍，有无找词困难，有无错语、新语、无意义杂乱及刻板言语，有无语法障碍；
  2. 听理解障碍：患者可听到声音，但对语义的理解不能或不完全。
  3. 复述
  4. 命名
  5. 阅读
  6. 书写

失用

失认

患者感觉通路正常，但不能经由某种感觉辨别熟识的物体，此种障碍并非由感觉、言语、智能和意识障碍引起。包括视觉失认、听觉失认和触觉失认。

视空间技能和执行功能

脑神经检查

嗅神经

让患者闭目，先后阻塞两侧鼻孔，用带有花香或其他气味的物质置于患者受检鼻孔。

视神经

视力

  1. 远视力：国际标准视力表；
  2. 近视力：标准视力表。

视野

  1. 周边视野检查：手动法（对向法）粗略测试、周边视野计精确测定。
  2. 中心视野检查。

眼底

动眼、滑车和展神经

  1. 外观：睑裂（对称性，是否有上睑下垂）
  2. 眼球运动
  3. 瞳孔及其反射：对光反射，调节及辐辏反射

三叉神经

  1. 面部感觉
  2. 咀嚼肌运动
  3. 反射：角膜反射、下颌反射。

面神经

  1. 面肌运动
  2. 感觉
  3. 反射：角膜反射、眼轮匝肌反射、掌骸反射
  4. 副交感神经纤维

前庭蜗神经

蜗神经

  1. 林纳（Rinne）试验：比较骨传导（BC）和气传导（AC）的听敏度；
    • 将震动的音叉（128Hz）置于耳后乳突部，听不到声音后迅速将音叉置于该侧耳旁，直至气传导听不见声音，再检查另一侧。
    • 正常结果：Rinne试验(+)——听到的时间AC＞BC
    • 传导性耳聋：BC＞AC，称Rinne试验(-)
    • 感音性耳聋：AC＞BC，但二者均缩短。
  2. 韦伯（Weber）试验：将震动的音叉置于患者额顶正中，比较双侧骨传导。
    • 正常结果：两耳感受到的声音相同
    • 传导性耳聋：患侧较响，称Weber试验(+)；
    • 感音性耳聋：健侧较响，称Weber试验(-)。

前庭蜗神经

观察自发性症状（眩晕呕吐眼球震颤平衡障碍），冷热水试验、转椅试验。

舌咽神经、迷走神经

  1. 运动检查：是否有声嘶、带鼻音或完全失音，双侧软腭抬举、腭垂偏斜，饮水呛咳
  2. 感觉
  3. 味觉
  4. 反射：咽反射、眼心反射、颈动脉窦反射

副神经

对侧转颈和同侧耸肩无力或不能，同侧胸锁乳突肌和斜方肌萎缩、垂肩和斜颈。

舌下神经

伸舌偏斜、舌肌萎缩、肌束颤动。

运动系统检查

肌容积

  1. 肌萎缩：下神经元损害、肌肉疾病
  2. 假性肥大：进行性肌营养不良（腓肠肌、三角肌）

肌张力

  1. 肌张力减低：肌肉弛缓柔软，被动运动阻力减低，关节活动范围扩大；
  2. 肌张力增高：肌肉较硬、被动运动阻力增加，关节活动范围缩小。
    • 折刀样肌张力增高：锥体束病变时表现为痉挛性肌张力增高，上肢屈肌和下肢伸肌肌张力增高明显，被动运动开始时肌张力增高，结束时变小。
    • 铅管样、齿轮样肌张力增高：锥体外系病变时表现为强直性肌张力增高，伸肌与屈肌张力均增高，向各方向被动运动时阻力均匀，也称为铅管样（不伴震颤）或齿轮样（伴震颤）。

肌力

肌力的六级记录法：

分级 表现
0级 完全瘫痪
1级 肌肉可收缩，但不能产生动作
2级 肌肉能在床面上移动，但不能抵抗自身重力，即不能抬起
3级 肢体能抵抗重力离开地面，但不能抵抗阻力
4级 肢体能做抗阻力动作，但不完全
5级 正常肌力

不自主运动

不能随意控制的舞蹈样动作、手足徐动、肌束颤动、震颤（静止性、运动性、姿势性）、肌张力障碍等。

共济运动

  1. 指鼻试验：嘱患者外展伸直一侧上肢，用示指尖触摸自己的鼻尖，睁眼和闭眼重复相同动作，注意两侧比较。
    • 辨距不良：小脑半球可见指鼻不准，接近目标时动作迟缓或出现动作（意向）性震颤，常超过目标（过指）。
  2. 反击征（Holmes反跳试验）：嘱患者收肩屈肘，前臂旋后、握拳，肘关节放于桌上或悬空靠近身体，检查者用力拉其腕部，受试者屈肘抵抗时，检查者突然松手。
    • 正常情况：屈肘运动立即停止，不会击中自己。
    • 小脑疾病：失去迅速调整能力，故屈肘力量会使前臂或掌部碰击自己的肩部或面部。
  3. 跟—膝—胫试验：仰卧位，上举一侧下肢，用足跟触及对侧膝盖，再沿胫骨前缘下移。
    • 小脑损害：抬腿触膝时出现辨距不良和一项项震颤，下移时摇晃不稳。
    • 感觉性共济失调患者：闭目时足跟难寻膝盖。
  4. 轮替试验：嘱患者用前臂快速旋前旋后，或一手用手掌、手背连续交替拍打对侧手掌，或用足趾反复拍打地面。
    • 轮替运动障碍：动作笨拙、节律慢而不协调。
  5. 起坐试验：取仰卧位，双手交叉置于胸前，不用支撑坐起。
    • 联合屈曲征：小脑病变患者髋部和躯干屈曲，双下肢向上抬离床面，起坐困难。
  6. 龙贝格（Romberg）征/闭目难立征：患者双足并拢站立，双手向前平伸、闭目，闭眼时出现摇摆伸直跌倒，称为龙贝格征阳性。

姿势与步态

感觉系统检查

  1. 浅感觉：痛觉、触觉、温觉；
  2. 深感觉：运动觉、位置觉、振动觉。
  3. 复合感觉：定位觉、两点辨别觉、图形觉、实体觉。

反射检查

深反射

反射 支配脊神经 传导神经
肱二头肌反射 $C_{5—6}$ 肌皮神经
肱三头肌反射 $C_{6—8}$ 桡神经
桡反射 $C_{5—8}$ 桡神经
膝反射 $L_{2—4}$ 股神经
踝反射 $S_{1—2}$ 胫神经
Hoffman征 $C_7—T_1$ 正中神经
Rossolimo征 $L_5—S_1$ 胫神经

阵挛：髌阵挛、踝阵挛。

浅反射

反射 支配脊神经 传导神经
腹壁反射 $T_7—T_{12}$ 肋间神经
提睾反射 $L_{1—2}$ 生殖股神经
跖反射 $S_{1—2}$ 胫神经
肛门反射 $S_{4—5}$ 肛尾神经

病理反射

  1. Babinski征
  2. Babinski等位征：足母趾背屈
    • Chaddock征：从外踝下方向前划至足背外侧；
    • Oppenheim征：用拇指和示指沿胫骨前缘自上向下用力下滑
    • Schaeffer征：用手挤压跟腱；
    • Gordon征：用手挤压腓肠肌
    • Gonda征：用力下压第4、5足趾，数分钟后突然放松；
    • 普谢普（Pussep）征：轻划足背外侧缘。
  3. 强握反射
  4. 脊髓自主反射

脑膜刺激征

  1. 颈强直：患者仰卧，检查者托患者枕部并使其头部前屈而表现不同程度的颈强直（被动屈颈受限）
  2. Kernig征：患者仰卧，下肢与髋、膝关节处屈曲成直角，检查者于膝关节处试行伸直其小腿。
    • 伸直受限并出现出现疼痛，大小腿间夹角＜135°，为Kernig(+)。
  3. Brudzinski征：患者仰卧屈颈时出现双侧髋、膝部屈曲，以及一侧下肢屈曲位，检查者使该侧下肢向腹部屈曲，对侧下肢亦发生屈曲，均Brudzinski征(+)。

自主神经检查

一般检查

皮肤黏膜、毛发和指甲、出汗、瞳孔

内脏及括约肌功能检查

自主神经反射检查

  1. 竖毛试验
  2. 皮肤划痕试验
  3. 眼心反射
  4. 血压和脉搏的卧立位试验
  5. 汗腺分泌发汗试验
