【神经病学】头痛

发表于2025-11-05|更新于2025-11-06|神经病学
|浏览量:

头痛是临床常见的症状，通常指局限于头颅上半部，包括眉弓、耳轮上缘和枕外隆凸连线以上部位的疼痛。

发病机制：①颅内外动脉的扩张；②颅内痛觉敏感结构组织被牵拉或移位（牵引性头痛）；③颅内外痛觉敏感组织炎症；④传导痛觉的脑神经和颈神经直接受压或炎症；⑤眼、耳、鼻、牙齿病变疼痛的扩散（牵涉性疼痛）。

分类（ICHD—3）：

  • 原发性疼痛：偏头痛、紧张性疼痛、三叉自主神经性疼痛、其他原发性疼痛。
  • 继发性疼痛：头颈部外伤、头颅和颈部血管疾病、非血管性颅内疾病、物质或物质戒断、感染、内环境紊乱、结构病变、精神疾病。
  • 痛性脑神经病、其他面痛与其他头痛与其他头痛。

偏头痛

偏头痛是常见的原发性头痛，其特征是具有反复发作性，多为单侧、中重度、搏动性疼痛，日常活动可加重，一般持续4—72小时，并伴有恶心和/或呕吐，畏光和畏声。

病因

  1. 遗传因素：遗传易感性
  2. 内分泌和代谢因素
  3. 饮食、药物和精神因素

发病机制

先兆期：神经元及胶质细胞去极化和超极化缓慢扩散的皮质扩散性抑制（CSD）相关。
头痛期：三叉神经血管系统激活。

临床表现

偏头痛分型

  • 无先兆偏头痛
  • 有先兆偏头痛
    • 典型先兆偏头痛：典型先兆伴头痛，典型先兆不办头痛；
    • 脑干先兆偏头痛；
    • 偏瘫性偏头痛：家族性偏瘫性偏头痛，散发性偏瘫性偏头痛；
    • 视网膜性偏头痛。
  • 慢性偏头痛
  • 偏头痛并发症：偏头痛持续状态，不伴脑梗死的持续先兆，偏头痛性脑梗死，偏头痛先兆诱发的癫痫发作。
  • 很可能的偏头痛：很可能的无先兆偏头痛，很可能的有先兆偏头痛。
  • 可能与偏头痛相关的周期综合症
    • 反复胃肠功能障碍：周期性呕吐综合征，腹型偏头痛；
    • 良性发作性眩晕；
    • 良性发作性斜颈。

偏头痛的几种主要亚型

无先兆偏头痛

最常见，发作频率高。

  1. 反复发作的一侧或双侧额颞部疼痛，呈搏动性，中重度疼痛；
  2. 伴随症状：恶心和/或呕吐、畏光和畏声。

有先兆偏头痛

先兆：部分患者在头痛之前常出现可逆的局灶性神经系统症状。

视觉先兆（双眼闪光、暗点或水波纹、视物变形）＞感觉先兆

  • 典型先兆偏头痛：完全可逆的视觉、感觉和/或言语症状；
  • 脑干先兆偏头痛：构音障碍、眩晕、耳鸣、听力减退、复视、共济失调、意识障碍。
  • 偏瘫性偏头痛：完全可逆的肢体无力，持续时间通常＜72h。
  • 视网膜性偏头痛：反复发生的完全可逆单眼视觉障碍。

慢性偏头痛

每月头痛发作超过15天，连续3个月或3个月以上，且每月至少有8天的头痛具有偏头痛特点。

偏头痛并发症

  1. 偏头痛持续状态
  2. 不伴脑梗死的持续先兆
  3. 偏头痛性脑梗死
  4. 偏头痛先兆诱发的癫痫发作

可能与偏头痛有关的周期综合征（儿童期周期性综合征）

诊断

无先兆偏头痛的诊断标准

  1. 符合下述2—4特征的头痛至少发作5次；
  2. 头痛发作持续4—72小时（未经治疗或治疗效果不佳）；
  3. 至少有下列4项中的2项：①单侧；②搏动性；③中、重度头痛；④日常体力活动可加重头痛或因头痛而避免日常活动；
  4. 发作过程换个至少符合下列2项中的1项：①恶心和/或呕吐；②畏光和畏声。
  5. 不能用ICHD—3中的其他诊断更好地解释。

有先兆偏头痛诊断标准

  1. 至少有2次发作符合下述2—3；
  2. 至少有1个可完全恢复的先兆症状：①视觉；②感觉；③言语和/或语言；④运动；⑤脑干；⑥视网膜。
  3. 至少符合下列6项中的3项：
    • ①至少有一个先兆持续超过5分钟；
    • ②2个或更多的症状连续发生；
    • ③每个独立先兆症状持续5—60分钟；
    • ④至少有一个先兆是单侧；
    • ⑤至少有一个先兆是阳性；
    • ⑥与先兆伴发或在先兆出现60分钟内出现头痛。
  4. 不能用ICHD—3中的其他诊断更好地解释。

慢性偏头痛诊断标准

  1. 符合2—3项的头痛每月发作至少15天，至少持续3个月。
  2. 符合无先兆偏头痛诊断2—4项标准和/或有先兆偏头痛2—3项标准至少发生5次。
  3. 头痛符合以下任何1项，且每月发作＞8天，持续时间＞3个月：
    • 符合无先兆偏头痛的3—4项标准；
    • 符合有先兆偏头痛的2—3项标准；
    • 患者认为的偏头痛可通过服用曲普坦类或麦角胺类药物缓解。
  4. 不能用ICHD—3中的其他诊断更好地解释。

鉴别诊断

  1. 丛集性头痛
  2. 紧张性头痛
  3. 继发性疼痛：对具有以下情况的头痛应谨慎排除：
    • 50y后的新发疼痛；
    • 血液高凝风险患者新发疼痛；
    • 有肿瘤或艾滋病史者新发疼痛；
    • 突然发生的、迅速到达高峰的剧烈头痛；
    • 与体位改变相关的头痛；
    • 伴有发热；
    • 伴有视神经乳头水肿、神经系统局灶症状和体征或认知障碍；
    • 头痛性质在短时间内发生变化。

治疗

治疗原则：减轻或终止头痛发作，缓解伴发症状，预防头痛复发。

发作期的治疗

疗效评价标准

  1. 2小时内无疼痛；
  2. 2小时内最困扰的伴随症状消失；
  3. 2小时后疼痛缓解；
  4. 治疗成功后的24小时内无头痛再发或镇痛药的使用。

药物

  1. 非特异性止痛药：对乙酰氨基酚和NSAIDs；
  2. 特异性药物：曲普坦类和降钙素基因相关肽（CGRP）受体拮抗剂、选择性$5-HT_{1F}$受体激动剂。

预防性治疗

适应症

  1. 每月2次以上的偏头痛发作；
  2. 急性期治疗无效或不能耐受；
  3. 存在药物过度使用风险；
  4. 严重影响生活、工作或学习；
  5. 存在频繁、持续时间较长或令患者极度不适的先兆；
  6. 特殊类型的偏头痛；
  7. 患者的自我要求。

药物选择

  1. β—肾上腺素受体拮抗剂；
  2. 抗癫痫药；
  3. 抗抑郁药；
  4. 血管紧张素Ⅱ受体拮抗剂（ARB）与血管紧张素转化酶抑制剂（ACEI）；
  5. CGRP类药物；
  6. 钙通道阻滞剂。

紧张型头痛

  1. 典型病例多在20岁左右发病，发病年龄高峰为25—30岁。
  2. 疼痛像一条带子紧束头部或呈颅周紧锢样、压迫样感，一般不伴有恶心、呕吐、畏光、畏声等症状。

丛集性头痛

是一种常见的三叉自主神经性头痛，表现为一侧眼眶周围发作性剧烈疼痛，有反复密集发作的特点，伴有同侧眼结膜充血、流泪、瞳孔缩小、眼睑下垂以及头面部出汗等自主神经症状和/或不安、躁动感。

临床表现

  1. 20—40岁，男性多见。
  2. 突然发生，无先兆症状，几乎发生于每日同一时间。
  3. 头痛每次发作几乎均为偏侧，呈尖锐、爆炸样、非搏动性剧痛。常伴有同侧颜面部自主神经功能症状。

治疗

急性期治疗

  1. 吸氧疗法（首选）；
  2. 曲普坦类药物；
  3. 利多卡因；
  4. 生长抑素及其类似物。

预防性治疗

维拉帕米、锂盐。

过渡性治疗（短期预防性治疗或桥接治疗）

非药物治疗（神经调控治疗）
