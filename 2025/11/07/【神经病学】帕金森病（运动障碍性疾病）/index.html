【神经病学】帕金森病（运动障碍性疾病）

发表于2025-11-07|更新于2025-11-07|神经病学
|浏览量:

帕金森病（PD）又名震颤麻痹，是一种常见于中老年的神经系统变性疾病，临床上以静止性震颤、运动迟缓、肌强直和姿势平衡障碍为主要特征。

病因与发病机制

病因

  1. 遗传因素
  2. 环境因素：①毒物：1—甲基—4—苯基—1,2,3,6-四氢吡啶（MPTP）、杀虫剂、除草剂等；②其他因素：吸烟、咖啡、NSAIDs等。
  3. 神经系统老化（促发因素）：30岁后，黑质多巴胺能神经元、纹状体多巴胺递质水平随年龄增长而逐渐减少。
  4. 多因素相互作用

发病机制

  1. 纹状体多巴胺水平显著降低：肌张力增高、动作减少等运动症状。
  2. 中脑—边缘系统系统和中脑—皮质系统的多巴胺水平显著降低：智能减退、情感障碍等高级神经活动异常。

病理

基本病变：

  1. 黑质致密区多巴胺能神经元及其他含色素的神经元大量变性丢失。
  2. 在残留的神经元胞质内出现嗜酸性包涵体，即路易体（Lewy body）。

临床表现

运动症状

  1. 静止性震颤：肢体完全静止状态时出现4—6Hz震颤，常为首发症状。
    • 静止时出现或增强，随意运动时减轻或消失，减轻或激动时加剧，入睡后消失。
    • 典型表现：拇指和屈曲的示指间呈“搓丸样”动作。
  2. 肌震颤：屈肌和伸肌张力同时增高。
    • 铅管样强直：四肢及颈部关节被动运动时始终保持阻力增高，似弯曲软铅管；
    • 齿轮样强直：如患者伴有震颤，检查者感觉在均匀阻力中出现断续停顿，如同转动齿轮。
  3. 运动弛缓：核心症状。
    • “面具脸”：查体见面容呆板、双眼凝视、瞬目减少。
    • 语速变慢、语音低沉、吐字欠清。
    • “小字征”：书写字体越写越小。
    • 做快速重复性动作时运动速度缓慢和幅度减小。
  4. 姿势步态与平衡障碍
    • “冻结”现象：PD患者在行走过程中步伐突然变小或停止，好似双脚突然站在地上，不能动弹。
    • 前冲步态/慌张步态：PD患者迈步后以极小的步伐越走越快，不能及时止步。

非运动症状

  1. 感觉障碍：早期嗅觉减退，中晚期肢体麻木、疼痛；
  2. 睡眠障碍：快速眼动睡眠行为障碍（RBD）、不宁腿综合征（RLS）；
  3. 自主神经功能障碍：便秘、多汗、脂溢性皮炎（油脂面），晚期性功能减退、排尿障碍、直立性低血压。
  4. 精神和认知障碍：伴有抑郁，其次伴有焦虑。

辅助检查

  1. 实验室检查：常规检查并无异常。少数患者发现基因突变。
  2. 嗅觉测试：嗅觉减退。
  3. 影像学检查：正电子发射断层扫描（PET）、单光子发射计算机断层扫描（SPECT）、心脏间碘苄胍（MIBG）照射技术、经颅超声。
  4. 病理：外周组织检查可见α—突触核蛋白异常累积。

诊断

必备条件

  1. 运动迟缓；
  2. 至少存在下列1项：肌强直或静止性震颤。

支持标准

  1. 多巴胺能药物治疗疗效明确且显著。
  2. 出现左旋多巴诱导的异动症。
  3. 既往或本次查体观察到单个肢体的静止性震颤。
  4. 以下辅助检测阳性有利于鉴别 帕金森病 和 非典型性帕金森综合征 ：
    • 存在嗅觉减退或丧失；
    • 或头颅超声显示黑质异常高回声（＞$20mm^2$）；
    • 心脏间碘苄胍（MIBG）闪烁现象法显示心脏去交感神经支配。

排除标准

  1. 存在明确的小脑性共济失调；
  2. 出现向下的垂直性核上性凝视麻痹，或者向下的垂直性扫视选择性减慢；
  3. 在发病后5年内，患者被诊断为高度怀疑的 行为变化型额颞叶痴呆 或 原发性进行性失语。
  4. 发病3年后仍局限于下肢的帕金森样症状；
  5. 多巴胺受体拮抗剂或多巴胺耗竭剂治疗诱导的帕金森综合征，其剂量与时程与药物性帕金森综合征一致；
  6. 患者对高剂量左旋多巴治疗缺乏显著治疗应答；
  7. 存在明确的皮质复合感觉缺失，以及存在明确的肢体观念运动性失用或进行性失语；
  8. 分子影像学检查突触前多巴胺能系统功能正常；
  9. 存在明确可导致帕金森综合征或疑似与患者症状相关的其他疾病，或者基于全面评估，由专业评估医师判断可能为其他综合症。

警示征象（支持诊断其他疾病）

  1. 发病后5年内出现快速进展的步态障碍，以至于需要经常使用轮椅；
  2. 运动症状或体征在发病后5年内或5年以上完全不进展，除非这种疾病的稳定与治疗相关；
  3. 发病后五年内出现球部功能障碍：严重发音困难、构音障碍或吞咽困难；
  4. 发病后5年内出现吸气性呼吸功能障碍；
  5. 发病后5年内出现严重的自主神经功能障碍：
    • 直立性低血压
    • 严重尿潴留或尿失禁，且不是简单的功能性尿失禁；
  6. 发病3年内由于平衡障碍导致反复跌倒（＞1次/年）；
  7. 发病后10年内出现不成比例的颈部前倾或手足挛缩；
  8. 发病后5年内不出现任何一种常见的非运动症状；
  9. 出现其他原因所不能解释的锥体束征；
  10. 起病或病程中表现为双侧对称性的帕金森综合征症状，没有任何侧别优势，且客观查体亦未观察到明显的侧别性。

诊断标准

临床确诊的帕金森病

  1. 不存在绝对排除标准；
  2. 至少存在两条支持性标准；
  3. 没有警示征象。

临床很可能的帕金森病

  1. 不存在绝对排除标准；
  2. 如果出现警示征象则需要通过支持性标准来抵消（1:1），如果出现2条以上警示征象则诊断不成立。

鉴别诊断

  1. 继发性帕金森综合征：继发于甲型脑炎后的帕金森综合征、药物性帕金森综合征、老年人基底节区多发腔隙性梗死、拳击手偶见头部外伤引起的帕金森综合征。
  2. 其他神经变性疾病的帕金森综合征。
  3. 其他：特发性震颤（无肌强直及动作性迟缓）、抑郁症、脑血管病。

治疗

治疗原则

  1. 综合治疗：药物治疗为首选和主要治疗手段，手术治疗为补充，肉毒毒素注射为治疗局部痉挛和肌张力障碍的主要手段。
  2. 用药原则
    • 总体目标：改善症状，提高工作能力和生活质量。
    • 强调个体化；
    • 坚持“剂量滴定”以避免产生药物急性副作用，力求实现“尽可能以小剂量达到满意临床效果”。

早期PD的药物治疗

老年前（＜65y）患者且不伴智能减退

  1. 非麦角类多巴胺受体（DR）激动剂（首选）：普拉克索、罗匹尼罗、吡贝地尔缓释片。
  2. B型单胺氧化酶（MAO—B）抑制剂（司来吉兰、雷沙吉兰），或加用Vit E。
  3. 复方左旋多巴（最基本、最有效药物）：多巴丝肼、卡左双多巴。
  4. 恩他卡朋双多巴片（恩他卡朋为儿茶酚—O—甲基转移酶(COMT)抑制剂）；
  5. 金刚烷胺。
  6. 抗胆碱能药：苯海索。

老年患者或伴智能减退

首选复方左旋多巴，必要时加用DR激动剂、MAO—B抑制剂或COMT抑制剂。

中晚期PD的药物治疗

运动并发症（症状波动和异动症）是中晚期患者最常见的症状，也是最棘手的治疗难题。

症状波动的治疗

  1. 疗效减退或剂末现象：每次用药有效时间缩短，症状随血药浓度波动而发生波动。
    • 处理方法：增加每日服药次数或增加每次服药剂量、改用缓释剂或加用MAO—B抑制剂或COMT抑制剂，也可加。
  2. “开—关”现象：症状在突然缓解（“开”）和加重（“关”）之间波动。“开期”伴异动症。
    • 处理方法：应用DR激动剂、持续皮下注射阿扑吗啡或左旋多巴肠凝胶灌注、脑深部电刺激（DBS）手术治疗。

异动症

常表现为不自主的舞蹈样、肌张力障碍样动作，可累计头面部、四肢、躯干。

  1. 剂峰异动症：常出现在血药浓度高峰期，与用药过量或多巴胺受体超敏有关；
    • 减少复方左旋多巴剂量。
  2. 双向异动症：发生于剂初和剂末。
    • 若在使用复方左旋多巴控释剂：换用常释剂，最好换用水溶剂（剂初异动症）
    • 加用长半衰期的DR激动剂或COMT抑制剂（缓解剂末异动症、改善剂初异动症）
  3. 肌张力障碍：足或小腿痛性肌挛。
    • 睡前服复方左旋多巴缓释剂或DR激动剂；
    • 在起床前服用左旋多巴水溶剂或常释剂。

非运动症状的治疗

  1. 感觉障碍：“开—关”现象相关疼痛或麻木可调整治疗，延长“开期”。
  2. 睡眠障碍：氯硝西泮或褪黑素
  3. 自主神经功能障碍
  4. 精神障碍

手术治疗

  • 神经核毁损术
  • 脑深部电刺激（DBS）——首选
