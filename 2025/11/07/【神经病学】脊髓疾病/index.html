【神经病学】脊髓疾病

发表于2025-11-07|更新于2025-11-09|神经病学
|浏览量:

脊髓损害分为不完全性脊髓损害和脊髓横贯性损害。前者表现为损害部位的相应症状和体征；后者表现为受累节段以下双侧上运动神经源性瘫痪、全部感觉缺失和括约肌功能障碍。

脊髓休克： 严重横贯性损害的急性期呈现脊髓休克，表现为弛缓性瘫痪，一般持续2—4周后转变为痉挛性瘫痪。

急性脊髓炎

急性脊髓炎是指各种感染后自身免疫反应所致的急性横贯性脊髓炎性改变，又称急性横贯性脊髓炎，是临床上最常见的一种脊髓炎。

注意：①急性脊髓炎的诱因是感染，但属于非感染性疾病，这是由其发病机制（自身免疫）决定的！②急性脊髓炎属于炎性病变。

临床表现

急性起病，首发症状多为双下肢麻木、无力和二便障碍，可以出现病变部位根痛和束带感。

  1. 运动障碍：早期可出现脊髓休克，一般持续2—4周进入恢复期，肌力恢复常始于下肢远端，逐步向上恢复。
  2. 感觉障碍：病变以下节段所有感觉障碍，感觉平面上缘可有感觉过敏或束带感。
  3. 自主神经功能障碍：尿失禁（多数）、尿潴留
  4. 上升性脊椎炎

辅助检查

  1. 脑脊液检查：细胞数正常或轻度升高，以L为主，蛋白含量正常或轻度升高。
  2. 影像学检查：MRI检查最有意义，急性期可见病变节段脊髓增粗、肿胀，斑片状长T1、长T2异常信号。
  3. 电生理检查：视觉诱导电位（VEP）、下肢体感诱发电位（SEP）、运动诱发电位（MEP）。

鉴别诊断

  1. 视神经脊髓炎谱系疾病：视力下降或VEP异常，视神经病变可在脊髓病变之前。
  2. 脊髓血管病：
    • 缺血性：12—24小时内达峰，深感觉保留。
    • 出血性：可出现血型脑脊液，MRI检查有助于诊断。
  3. 急性脊髓压迫症：脊柱影像学检查可见椎体破坏、椎间隙变窄或椎体寒性脓肿。
  4. 急性硬脊膜外脓肿：有化脓性病灶及感染病史。
  5. 吉兰—巴雷综合征：瘫痪程度远端重于近端，呈末梢型感染障碍。

治疗

药物治疗

  1. 糖皮质激素
  2. IVIg
  3. 抗感染药物
  4. B族维生素
  5. 其他：血管扩张药、神经营养药。

康复治疗

护理

预后

如无严重并发症，多于3—6月内恢复生活自理。

完全性截瘫6月后肌电图仍为失神经改变，MRI显示病灶范围大且伴软化灶形成者预后不良。

脊髓压迫征

脊髓压迫征是一组椎管内或椎骨占位性病变所引起的脊髓受压综合征，随病变进展出现脊髓半切综合征、横贯性损害及椎管梗阻，脊神经根和血管可不同程度受累。

临床表现

急性脊髓压迫征

急性发病、进展迅速，数小时至数日内达峰。

脊髓休克：病变水平以下弛缓性瘫痪、各种感觉及反射消失、尿便障碍。

慢性脊髓压迫征

分期

  1. 根痛期：神经根痛及脊膜刺激症状；
  2. 脊髓部分受压期：脊髓半切综合征；
  3. 脊髓完全受压期：脊髓完全横贯性损害；

神经根症状

主要表现为根痛和局限性运动障碍。

  1. 根痛：剧烈，常被描述为电击样、烧灼样、刀割样或撕裂样疼痛。
  2. 局限性运动障碍：前根受累早期可见肌束颤动，后期可出现肌无力和萎缩。

感觉障碍

  1. 脊髓丘脑束受累：比病变低2—3个节段的对侧躯体的痛、温觉减退或缺失。
  2. 后索受累：病变水平以下同侧深感觉减弱或消失。
  3. 髓外病变：感觉障碍自下肢远端向上发展至受压节段。
  4. 髓内病变：早期可出现病变节段支配区分离性感觉障碍，累及脊髓丘脑束时感觉障碍自病变节段向下发展，鞍区（$S3—5$）感觉保留至最后受累，称为 “马鞍回避” 。

运动障碍

  1. 一侧锥体束受累：同侧肢体痉挛性瘫痪、肌张力增高、腱反射亢进、病理征阳性。
  2. 双侧锥体束受累：双下肢痉挛性瘫痪，初期伸直样，晚期屈曲样。
  3. 脊髓前角及前根受累：支配肌群弛缓性瘫痪伴肌束震颤和肌萎缩。

辅助检查

脑脊液检查

  1. 压颈试验：压颈时压力上升较快、解除压力后下降较慢，或上升慢下降更慢提示不完全梗阻。
  2. Froin综合征：椎管严重梗阻时脑脊液蛋白—细胞分离，即细胞数正常，蛋白含量超过10g/L时，黄色的脑脊液流出后自动凝结。

影像学检查

  1. 脊柱X线检查；
  2. CT及MRI检查（首选）；
  3. 脊髓造影。

诊断

  1. 明确脊髓损害为压迫性或非压迫性；
  2. 确定受压部位及平面、病变位于髓内、髓外硬膜内还是硬膜外；
  3. 脊髓受压程度；
  4. 确定病因或性质。

鉴别诊断

  1. 急性脊髓炎：多有感染及预防接种史，数小时至数日达峰，急性期腰穿动力学实验一般无梗阻。
  2. 脊髓空洞症：病损节段支配区皮肤分离性感觉障碍，MRI显示脊髓内长条形空洞有助于鉴别。
  3. 亚急性联合变性。

治疗

  1. 病因治疗
  2. 防止并发症和对症治疗
  3. 康复治疗
