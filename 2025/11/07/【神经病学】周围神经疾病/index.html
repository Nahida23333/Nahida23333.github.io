【神经病学】周围神经疾病

发表于2025-11-07|更新于2025-11-08|神经病学
|浏览量:

周围神经病是指各种病因以前你的周围神经系统结构或者功能损害的疾病总称。

  • 脑神经疾病：原发性三叉神经痛、面神经痛；
  • 脊神经疾病：正中神经麻痹、尺神经麻痹、桡神经麻痹、腓总神经麻痹、坐骨神经痛、吉兰—巴雷综合征

（原发性）三叉神经痛

表现为三叉神经分布区内短暂的反复发作性剧痛。

病因

周围学说、中枢学说。

发病机制

初级神经元的敏化和离子通道的改变。

临床表现

  1. 40岁以后发病，多女性。
  2. 骤然发生的剧烈的闪电样、刀割样疼痛。
  3. 大多单侧受累，疼痛严格限于三叉神经感觉支配区；
  4. “扳机点”或“触发点”：患者口角、鼻翼、颊部或舌部为敏感区，轻触可诱发。
  5. “痛性抽搐”：严重病例可因疼痛出现面肌反射性抽搐，口角牵向患侧。
  6. 神经系统查体多无阳性体征。

辅助检查

  1. 神经电生理检查
    • V1反射：电刺激三叉神经眼支出现瞬目反射；
    • V2、V3反射：电刺激三叉神经上颌支、下颌支出现咬肌抑制反射。
  2. 影像学检查：MRI检查、三叉神经磁共振断层血管成像；

鉴别诊断

继发性三叉神经痛、牙痛、舌咽神经痛

治疗

  1. 药物治疗：首选卡马西平或奥卡西平；
  2. 神经阻滞疗法：肉毒毒素、无水乙醇、甘油
  3. 经皮半月神经节射频电凝疗法
  4. 手术治疗：三叉神经显微血管减压术。

特发性面神经麻痹/贝尔麻痹/面神经炎

因茎乳突内面神经特异性炎症而导致周围性面瘫，为面瘫最常见的原因。

病因及发病机制

嗜神经病毒感染及感染后免疫反应，面神经缺血水中而导致受压。

临床表现

  1. 多见于20—40岁，男性。
  2. 急性起病，3天左右达峰。
  3. 主要表现：单侧周围性面瘫（受累侧闭目、皱眉、鼓腮、示齿、闭唇无力），以及口角歪向对侧；可伴有同侧耳后疼痛或乳突压痛。
  4. 体格检查：
    • 贝尔征：患侧闭眼时眼球向外上方转动，露出白色巩膜。
    • 鼻唇沟变浅、口角下垂，露齿时口角歪向健侧；
    • （口轮匝肌瘫痪）：鼓气、吹口哨漏气。
  5. 不同部位面神经损害出现不同临床症状。
    • 膝状神经节前损害：鼓索神经受累（同侧舌前2/3味觉障碍）、镫骨肌分支受累（听觉过敏、过度回响）；
    • 膝状神经节病变：除面神经麻痹、舌前2/3味觉障碍、听觉过敏，还有耳廓、外耳道感觉减退和外耳道、鼓膜疱疹，称 【亨特综合征】 。
    • 茎乳突附近病变：上述典型面瘫体征和耳后疼痛。

鉴别诊断

  1. 吉兰—巴雷综合征
  2. 耳源性面神经麻痹
  3. 颅后窝肿瘤或脑膜炎
  4. 神经系统莱姆病

治疗

  1. 药物治疗：糖皮质激素（尽早使用）、B族维生素、抗病毒治疗（Hunt综合征）。
  2. 护眼；
  3. 康复。

正中神经麻痹

病因

  1. 腕部正中神经嵌压伤（最常见）：腕管综合症（CTS）；
  2. 腕部锐器戳伤或被利器切割伤；
  3. 肱骨前臂骨折及穿透伤；
  4. 肘前区静脉注射时药物外渗；
  5. 肩关节脱位。

临床表现

  1. 运动障碍：握力及前臂旋前功能障碍。
    • 前臂旋前不能，腕外展屈曲不能，拇、示、中指不能屈曲，握拳无力，拇指不能对掌、外展及屈曲。
    • 肌肉萎缩犹以大鱼际明显，手掌扁平。
    • 拇指内收呈“猿手”畸形；
  2. 感觉障碍：手掌面桡侧，中指和无名指的桡侧一半；手背面示、中指末节和无名指末节桡侧半。
  3. 常见的综合征——腕管综合症：各种内科疾病致腕管内容物水肿、静脉淤滞，手腕部反复用力或创伤等原因导致正中神经在腕管内受压，出现桡侧三指感觉异常、麻木及大鱼际肌萎缩。

尺神经麻痹

病因

嵌压、外伤、炎症、骨折、麻风。

肘管综合症：尺神经在肘部肱骨内上髁后方、尺骨鹰嘴及肘内侧韧带构成的狭窄通道走行，加之位置表浅，故肘管内时最常见的嵌压部位，称为肘管综合症。

临床表现

  1. 运动障碍：屈腕困难，手向桡侧偏斜，“爪形手”。
  2. 感觉障碍：手掌和手背的尺侧，整个小指和无名指的尺侧一半。
  3. 肘管综合症：掌侧和尺侧一个半手指麻木，第一背侧脊神经和小鱼际肌萎缩。

桡神经麻痹

常见卡压部位：上臂桡神经沟处。

临床症状

  1. 高位损伤（腋部）：上肢各伸肌完全瘫痪，肘、腕、掌指关节均不能伸直，前臂伸直位旋后不能，手通常属于旋前位。
  2. 桡神经沟损伤（上臂）（最常见）：不合适的睡眠体位，多于晨起时发现垂腕垂指，可伴有手背及前臂桡侧麻木。（星期六夜麻痹）
  3. 后骨间神经损伤（前臂）：典型垂指而不垂腕，腕背伸向桡侧偏斜且不伴有感觉障碍；
  4. 桡浅神经损伤（前臂）：感觉障碍仅限于“虎口区”皮肤（手背拇指与第1、2掌骨间隙背侧）

腓总神经疼痛

表现为跨阈步态。

坐骨神经痛

坐骨神经痛时沿坐骨神经通路及其分支区内分布的疼痛综合征。

病因

  1. 继发性坐骨神经痛：周围组织或病变压迫、刺激；
  2. 根性坐骨神经痛：较干性坐骨神经痛多见，以腰椎间盘突出引起者最为多见。

临床表现

  1. 青壮年多见，单侧居多。
  2. 疼痛沿坐骨神经径路向腰部、臀部向股后、小腿后外侧和足外侧放射。
  3. 持续性钝痛继发性加剧，或为电击、刀割、烧灼样痛，行走和牵拉坐骨神经时疼痛明显。
  4. 查体可发现直腿抬高试验[拉塞格(Lasegue)征]阳性、直腿抬高加强试验阳性。
  5. 患侧小腿和足背可出现感觉障碍；踝反射减弱或消失。
  6. 影像学有助于查明病因，神经痛节段电生理可无阳性发现。

治疗

  1. 病因治疗；
  2. 药物治疗：神经营养剂、止痛剂、地西泮；
  3. 封闭疗法
  4. 物理疗法：超短波、红外线（急性期），感应电、碘离子透入、热疗（疼痛减轻后）。
  5. 手术治疗

吉兰—巴雷综合征（GBS）

GBS是一种自身免疫介导的多发性脊神经根神经病，典型临床表现为急性起病，四肢弛缓性无力，常有脑脊液蛋白—细胞分离现象，多呈单时相、自限性病程，IVIg和PE治疗有效。

病因与发病机制

自身免疫、分子模拟。

分型

急性炎性脱髓鞘性多发性神经根神经病（AIDP）

病理为多发神经根和周围神经节段性脱髓鞘。

临床表现

  1. 任何年龄、任何季节均可发病；
  2. 前驱事件：发病前4周内呼吸道/胃肠道感染史或疫苗接种史。
  3. 病程特点：急性起病，多在7天—2周左右达峰，多为单相病程，病程中可有短暂波动。
  4. 主要症状和体征：肢体对称性弛缓性肌无力（核心症状），自远端向近端发展或自近端向远端加重。
  5. 感觉障碍：手套—袜套样分布。
  6. 脑神经受累：双侧面神经麻痹最常见。
  7. 自主神经功能障碍：部分可有，表现为皮肤潮红、出汗增多、心动过速、心律失常、直立性低血压、手足肿胀、营养障碍、尿便障碍。

辅助检查

  1. 脑脊液检查
    • 脑脊液蛋白—细胞分离（GBS特征）
    • 寡克隆区带（OB）（非特征性改变）
    • 抗神经节苷脂抗体阳性（部分患者）
  2. 血清学检查
    • 抗神经节苷脂抗体阳性（部分患者，阳性率高于脑脊液）
    • 抗GM1抗体阳性（部分，特异性低）
  3. 粪便分离培养：空肠弯曲菌；
  4. 神经电生理：远端潜伏期延长、传导速度减慢，F波可见传导速度减慢或出现率下降。
  5. 腓肠神经活检（不作为必需检查）：有髓纤维脱髓鞘，部分出现吞噬细胞浸润，小血管周围可有炎症细胞浸润。

诊断标准

  1. 常有前驱感染史，急性起病且进行性加重，多2—4周内达峰。
  2. 对称性肢体和脑神经支配肌肉无力，重症可有呼吸机无力，四肢腱反射减弱或消失；
  3. 可伴轻度感觉异常和自主神经功能障碍；
  4. 脑脊液：蛋白—细胞分离；
  5. 神经电生理：远端潜伏期延长、传导速度减慢，F波异常、传导阻滞、异常波形离散等。
  6. 病程自限性。

鉴别诊断

出现下列症状不支持GBS诊断：

  1. 持续进展肢体无力＞8周，或呈不对称性；
  2. 以膀胱或直肠功能障碍为首发症状，或持久的膀胱和直肠功能障碍；
  3. 脑脊液单核细胞数＞$50×10^6/L$；
  4. 脑脊液出现分叶核白细胞；
  5. 存在明确的感觉平面。

需要鉴别诊断的疾病：急性起病的慢性炎性脱髓鞘性多发性神经根神经病（A—CIDP）、脊髓灰质炎、急性横贯性脊髓炎、低钾性周期性瘫痪、重症肌无力（MG）。

急性运动轴突性神经病（AMAN）

儿童更常见，男女发病率相似，国内夏秋季发病多。

病理以广泛的运动脑神经、脊神经前根及运动神经纤维轴索病变为主。

诊断参考AIDP，突出特点是神经电生理检查提示近乎纯运动神经受累。

急性运动感觉轴突性神经病（AMSAN）

以广泛神经根和周围神经的运动以及感觉纤维的轴索变性为主。

诊断参考AIDP，突出特点是神经电生理检查提示感觉和运动神经轴索损害明显

米勒—费希尔综合征（MFS）

以眼外肌麻痹、共济失调和腱反射消失为主要临床特点。

诊断标准

  1. 急性起病，病情在数天内或数周内达到高峰；
  2. 临床上以眼外肌麻痹、共济失调和腱反射消失为三大主要临床症状（部分患者可只有2项）；
  3. 脑脊液检查：蛋白—细胞分离；
  4. 病程呈自限性。

急性泛自主神经病（APN）

急性感觉神经病（ASN）

治疗

  1. 免疫治疗：IVIg、PE、糖皮质激素（不推荐）；
  2. 呼吸机麻痹的处理：保持呼吸道通畅和通气功能良好（翻身拍背、监护、气管插管或气管切开，机器辅助通气）；
  3. 对症、支持治疗：营养支持，预防控制坠积性肺炎和尿路感染等。
  4. 神经营养和康复治疗：B族维生素。

预后

自限性。
