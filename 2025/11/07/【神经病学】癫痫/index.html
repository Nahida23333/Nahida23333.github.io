【神经病学】癫痫

发表于2025-11-07|更新于2025-11-07|神经病学
|浏览量:

癫痫发作：由不同病因引起的脑部神经元高度同步化异常放电所导致的脑功能失调。

癫痫：有着不同病因基础、临床表现各异但以反复癫痫发作为共同特征的慢性脑部疾病状态。

癫痫综合征：由特定病因所致，具有特征性临床和脑电图表现的癫痫。

病因与发病机制

  1. 结构性病因：皮质发育不良、肿瘤、头外伤、脑血管疾病；
  2. 感染性病因：颅内感染；
  3. 遗传性病因：Dravet综合征；
  4. 代谢性病因：缺血缺氧性脑病、甲状旁腺减退、尿毒症脑病；
  5. 免疫性病因：自身免疫性脑炎、桥本脑病、神经精神狼疮。
  6. 不明原因。

发病机制

  1. 离子通道异常；
  2. 异常网络重组；
  3. 脑电图上癫痫样放电。

分类

  • 全面性起源
    • 运动性：肌阵挛—强直—阵挛发作（拆出来每一个也算）、失张力发作、癫痫性痉挛发作。
    • 非运动性：典型失神发作、非典型失神发作等。
  • 局灶性起源（意识清楚或意识受损）
    • 运动性：自动症、失张力发作、阵挛发作等。
    • 非运动性：自主神经性发作、行为中止。
  • 起源不明
    • 运动性：强直—阵挛发作、癫痫性痉挛发作。
    • 非运动性：行为中止。
    • 不能归类

临床表现

发作特征

  1. 发作性：症状突然发生，持续一段时间后迅速恢复，间歇期正常；
  2. 短暂性：发作持续时间非常短，通常为数秒钟或数分钟；
  3. 重复性：第一次发作后，经过不同间隔时间会有第二次或更多次的发作；
  4. 刻板性：每次发作的临床症状几乎一致。

癫痫发作的临床表现

全面性起源的癫痫发作

最初的症状学和脑电图表现提示发作起源于双侧脑部，且在双侧脑内扩布。

运动性：有明显运动症状的癫痫发作

  1. 全面性强直—阵挛发作（GTGS）：意识丧失、双侧肢体强直然后紧跟有阵挛的序列活动是其主要临床特征。
    • 强直期：全身骨骼肌持续性收缩。
    • 阵挛期：肌肉交替性收缩与松弛，呈一张一弛交替性抽动，阵挛频率逐渐变慢、间歇期延长。
    • 发作后期：尚有短暂阵挛（以面肌和咬肌为主）；全身肌肉松弛，可有尿失禁；呼吸、瞳孔、血压、心率依次恢复；肌张力降低，意识逐渐恢复。
  2. 强直发作和阵挛发作：上述发作仅有强直或阵挛发作。
  3. 肌阵挛发作：快速、短暂、触电样肌肉收缩，可遍及全身，也可限于某个肌群或某个肢体，常成簇发生，声、光等刺激可诱发，发作时间短暂，可无意识障碍。（脑电图：多棘慢波）
  4. 失张力发作：肌张力突然丧失，可致头呈点头样下垂或肢体下垂，重者可致跌倒。（脑电图：多棘慢波或低电位活动）

非运动性：无明显运动症状的癫痫发作

  1. 典型失神发作：儿童期起病，表现为原有活动突然停止，发呆、呼之不应，持续5—20秒，一般不跌倒，手中物体落地。发作后立即清醒，无明显不适，可继续先前活动，醒后不能回忆。
  2. 不典型失神发作：发作缓慢，除意识丧失外常伴肌力降低，偶有肌阵挛。（较慢的(2.0—2.5Hz)不规则棘慢波或尖慢波）

局灶性起源的癫痫发作

  1. 分为意识清楚和意识受损的局灶性发作。
  2. 根据发作期是是否存在明显运动症状，分为运动性和非运动性。

运动性：有明显运动症状的癫痫发作

  1. 过度运动性综合征：多动性运动，包括剧烈的摆动或蹬腿运动；
  2. 自动症：意识受损和看起来有目的、但实际上没有目的的发作性行为障碍；
  3. 托德（Todd）瘫痪或Todd麻痹：局灶性起源的运动性癫痫发作严重者可留下短暂性肢体瘫痪。
  4. Jackson发作：异常运动从局部起始沿皮质功能区传播，如从手指—腕部—前臂—肘—肩—口角—面部逐渐发展；
  5. 旋转性发作：双眼突然向一侧偏斜，继之头部发生不自主同向转动，伴有身体的扭转，但很少超过180°，部分患者过度旋转以至于跌倒。

非运动性：无明显运动症状的癫痫发作

  1. 感觉性发作
  2. 自主神经性发作
  3. 情绪性发作
  4. 行为中止
  5. 认知性发作

不明起源的癫痫发作

诊断

首选明确是否为癫痫发作，随后明确其发作类型，最后区分其是否为特定的癫痫综合征，同时考虑可能的病因和共患病。

传统上临床出现2次（间隔至少24小时）非诱发性癫痫发作即可诊断癫痫。

常用的辅助检查有脑电图（EEG）、神经影像学检查等，以EEG最为重要。

鉴别诊断

  1. 假性癫痫发作/癔症样发作/心因性非癫痫发作（PNES）：心理障碍而非脑电紊乱引起的脑部功能异常。发作时脑电图上无相应的癫痫样放电，抗癫痫发作治疗无效。
  2. 晕厥：弥漫性脑部缺血缺氧。
  3. 偏头痛
    癫痫发作 偏头痛
    头痛特点 程度轻，发作前后出现 偏侧或双侧剧烈头痛
    脑电图 阵发性棘波或棘慢波 局灶性慢波或正常
    视幻觉 简单视幻觉、复杂视幻觉 简单视幻觉
    意识障碍 发生突然、很快终止，程度重 发生缓慢，易唤醒
  4. 短暂性脑缺血发作；
  5. 低血糖症：血糖水平＜2mmol/L，常见于胰岛细胞瘤或长期服药的T2DM患者。

选择正确用药的时机

诊断一旦明确，除一些良性的癫痫综合征外，都应该立即开始用药。

癫痫持续状态（SE）

定义

  1. $t_1$：持续癫痫发作活动应被视为异常延长且应启动治疗的时间点。
  2. $t_2$：持续癫痫发作活动可能造成显著的长期不良成果。

治疗

治疗原则

  1. 治疗目标是尽快终止临床发作和电发作；
  2. 尽早治疗，遵循SE处理流程，尽快终止发作；
  3. 积极查找SE病因，对因治疗；
  4. 支持治疗：维持患者呼吸、循环及水、电解质平衡。

惊厥性SE

  1. 院前治疗：咪达唑仑、地西泮；
  2. 院内治疗
    • 初始治疗药物：苯二氮䓬类药物
    • 第二阶段：二线药物
    • 第三阶段：全身麻醉药
