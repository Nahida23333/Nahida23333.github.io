【神经病学】神经肌肉接头疾病和肌肉疾病

发表于2025-11-08|更新于2025-11-08|神经病学
|浏览量:

神经肌肉接头疾病是指神经肌肉接头传递障碍所引起的疾病，包括重症肌无力、Lambert—Eaton肌无力综合征及先天性肌无力综合征等。

发病机制：

  1. 神经肌肉接头疾病
    • 突触前膜的ACh合成和释放减少：Lambert—Eaton肌无力综合征、肉毒杆菌菌素。
    • 突触间隙中的AChE活性降低：有机磷；
    • 突触后膜的AChR数量或结构异常：重症肌无力。
  2. 肌肉疾病
    • 肌膜电位异常：周期性瘫痪等；
    • 能量代谢异常：线粒体肌病等；
    • 肌膜内病变：肌营养不良。

临床症状与体征：

  1. 肌无力；
  2. 肌肉萎缩；
  3. 运动不耐受；
  4. 肌肉肥大与假性肌肉肥大：
    • 病理性肌肉肥大：肌肉组织受到病变的刺激或代偿性增生而出现的肌肉肥大，如先天性肌强直；
    • 假性肌肉肥大：局部组织中脂肪组织与结缔组织增生而导致的肌肉肥大，见于假肥大性肌营养不良。
  5. 肌肉疼痛：神经末梢受刺激；
  6. 肌束不自主运动：
    • 肌束震颤/束颤：一个或多个运动单位所支配肌纤维的不自主收缩，不能引起关节运动；
    • 肌纤维震颤/纤颤：一条或数条肌纤维的收缩，能在肌电图上显示，一般肉眼不能观察；
    • 肌纤维颤搐：一组肌肉呈蠕动样运动，常有局部异常、不适或酸痛感。

重症肌无力（MG）

MG是一种神经肌肉接头传递障碍引起的获得性自身免疫性疾病，临床表现为骨骼肌无力和易疲劳，活动后症状加重，休息和应用胆碱酯酶抑制剂治疗后症状明显减轻。

病因与发病机制

  1. AChR抗体直接竞争性抑制或间接 干扰ACh和AChR的结合 ；
  2. 与AChR结合的AChR抗体通过激活补体而使 AChR降解和结构改变 ；
  3. AChR的内吞机制使突触后膜上的 AChR绝对数目减少、突触后膜褶皱破坏 。

病理

  1. 胸腺：重量增加、淋巴滤泡增生，生发中心增多；
  2. 神经肌肉接头：突触间隙加宽、突触后膜褶皱变浅且数量减少；
  3. 肌肉组织：少数肌肉组织的肌内衣小血管周围可见淋巴细胞浸润（“淋巴漏”）。

临床表现

两个高峰：20—40岁（女性多见），40—60岁（男性多见），多合并胸腺瘤。

临床特征

  1. 受累骨骼肌病态疲劳：连续收缩后无力，休息后缓解。
    • “晨轻暮重”：下午或傍晚肌无力加重，晨起或休息后减轻的波动现象。
  2. 受累肌的分布和临床表现
    • 首发症状：一侧或双侧眼外肌麻痹，出现上睑下垂、复视。
    • 累及面部肌肉和口咽肌：表情淡漠、苦笑面容、说话鼻音，咀嚼无力、饮水呛咳、吞咽困难；
    • 胸锁乳突肌和斜方肌：颈软、抬头困难、转颈耸肩无力；
    • 四肢肌肉受累以近端为重，腱反射通常不受影响。
  3. 重症肌无力征象：指呼吸机受累时出现咳嗽无力伸直呼吸困难，需要呼吸机辅助通气，使主要死亡原因。
    • 肌无力危象：疾病进展严重导致的呼吸肌麻痹性呼吸困难，新斯的明有显著疗效。
    • 胆碱能危象：应用AChE抑制剂过量引起的呼吸困难，常伴有瞳孔缩小、汗多、唾液分泌增多等副作用。新斯的明无效，反而加重病情。
  4. 特殊类型MG
  5. 胆碱酯酶抑制剂治疗有效；
  6. 病程特点：亚急性发病，病程有波动，缓解与复发交替，持续时间长。

MGFA临床分型

  • Ⅰ型：眼肌无力，可伴闭眼无力，其他肌群肌力正常；
  • Ⅱ型：除眼肌外的其他肌群轻度无力，可伴眼肌无力
    • Ⅱa型：主要累及四肢肌 和/或 躯干肌，可有较轻的咽喉肌受累；
    • Ⅱb型：主要累及咽喉肌 和/或 呼吸机，可有轻度或相同的四肢肌 和/或 躯干肌受累。
  • Ⅲ型：除眼肌外的其他肌群中度无力，可伴任何程度的眼肌无力；
    • Ⅱa型：主要累及四肢肌 和/或 躯干肌，可有较轻的咽喉肌受累；
    • Ⅱb型：主要累及咽喉肌 和/或 呼吸机，可有轻度或相同的四肢肌 和/或 躯干肌受累。
  • Ⅳ型：除眼肌外的其他肌群重度无力，，可伴任何程度的眼肌无力；
    • Ⅱa型：主要累及四肢肌 和/或 躯干肌，可有较轻的咽喉肌受累；
    • Ⅱb型：主要累及咽喉肌 和/或 呼吸机，可有轻度或相同的四肢肌 和/或 躯干肌受累。
  • Ⅴ型：气管插管，伴或不伴机械通气（除术后常规使用）
    • Ⅴb型：仅鼻饲而不进行气管插管。

辅助检查

  1. 疲劳试验：嘱患者持续上视出现上睑下垂，或两臂持续平举出现上臂下垂，休息后恢复为阳性。
  2. 胆碱酯酶抑制剂试验：新斯的明试验最常用。
  3. 神经肌肉电生理检查：
    • 重复神经刺激（RNS）：动作电位波幅递减，且低频刺激第4或第5波的波幅相对于第1波衰减10%—15%以上，高频刺激递减程度在30%以上。
    • 常规肌电图和神经传导速度（一般正常）
    • 单纤维肌电图（SFEMG）：颤抖增宽和/或阻滞。
  4. 抗体健侧：AChR抗体（滴度与症状不成比例）、MuSK抗体、LRP4抗体；
  5. 胸腺CT或MRI扫描检查。

鉴别诊断

Lambert—Eaton肌无力综合征、肉毒杆菌中毒、进行性延髓麻痹、慢性进行性眼外肌麻痹。

治疗

药物治疗

  1. 胆碱酯酶抑制剂：溴吡斯的明（最常用）
  2. 糖皮质激素：递增法、递减法（起效快但症状严重）
  3. 免疫抑制剂：硫唑嘌呤、环孢素A、吗替麦考酚酯、他克莫司、环磷腺苷、甲氨蝶呤、生物制剂；
  4. 禁用和慎用药物：氨基糖苷类等。

    部分患者在开始激素治疗的短期内可能出现病情加重，甚至出现肌无力危象。
    大剂量和长期应用糖皮质激素可诱发糖尿病、股骨头坏死、胃溃疡出血、严重继发感染、Cushing综合征等。

胸腺治疗

  1. 伴胸腺瘤MG：手术切除必需。
  2. 不伴胸腺瘤MG：择期行胸腺切除术。

血浆置换（PE）和静脉注射大剂量免疫抑制剂（IVIg）

危象处理

  1. 立即进行气管插管或切开，保持呼吸道通畅；
  2. 肌无力危象者应加大AChE抑制剂用量，胆碱能危象者应暂停AChE的应用；
  3. 基本处理
    • 保持呼吸道通畅，加强排痰，以免发生窒息；
    • 积极控制感染，选用有效、足量且对神经肌肉接头无阻滞作用的抗生素以控制肺部感染。
    • 积极的PE和IVIg治疗。

低钾型周期性瘫痪

周期性瘫痪是以反复发作的骨骼肌弛缓性瘫痪为特征的一组肌病。低钾型周期性瘫痪是周期性瘫痪中最常见的类型，以发作性肌无力、血清钾降低、补钾后肌无力迅速缓解为特征。

病因与发病机制

原发性低钾型周期性瘫痪1型：编码骨骼肌细胞钙离子通道α—1亚单位的基因CACNA1S突变而致病。

原发性低钾型周期性瘫痪2型：编码钠离子通道NaV1.4蛋白α亚基的基因SCN4A突变而致病。

临床表现

  1. 任何年龄均可发病，20—40岁男性多见；
  2. 常于夜间睡眠或清晨起床时出现对称性肢体无力或完全瘫痪，下肢重于上肢、近端重于远端。在发病期，主要体征为肢体不同程度的瘫痪、肌张力低下、腱反射减弱或消失，但无病理反射。
  3. 发作一般经数小时至数日逐渐恢复，最先受累的肌肉最先恢复；发作频率不等，发作间期一切正常。

辅助检查

  1. 发作期[$K^+$]＜3.5mmol/L，发作间期正常；
  2. 心电图：低钾性改变，U波出现，T波低平或倒置，PR间期和QT间期延长，ST短下降，QRS波增宽。
  3. 肌电图：运动电位时限短、波幅低；如完全瘫痪，则运动单位电位消失，电刺激无反应；
  4. 基因检测；
  5. 甲功、血气、尿常规、血生化（与继发性鉴别）

鉴别诊断

  1. 周期性瘫痪不同类型之间的鉴别：血清钾水平、临床特点（如发病时间）、基因检测
  2. 继发性周期性瘫痪：甲亢、原发性醛固酮增多症、肾小管酸中毒、失钾性肾炎、腹泻、药源性等导致的发作性肌无力。
  3. 重症肌无力：脑神经支配肌肉多首先受累，有晨轻暮重的特征。
  4. 吉兰巴雷综合症：脑脊液蛋白—细胞分离现象。

Duchenne型肌营养不良（DMD）

进行性肌营养不良（PMD）是一组遗传性进行性骨骼肌变性疾病。临床特征为缓慢进行性、对称性肌无力和萎缩，血清CK升高。

DMD是我国最常见的X连锁隐性遗传肌病，男性多于女性。

临床表现

  1. “鸭步”：臀中肌无力时导致行走时骨盆向两侧上下摆动。
  2. 高尔征（特征性表现）：患者自仰卧位起立时必须先翻身为俯卧位，其次屈膝关节和髋关节，病用手支撑躯干成俯跪位，然后以双手和双腿共同支撑躯干，再用手按压膝部以辅助股四头肌的肌力，身体呈深鞠躬位，最后双手攀附四肢缓慢地站立。
  3. 翼状肩胛：因前锯肌和斜方肌萎缩无力，举臂时肩胛骨内侧远离胸壁，两肩胛骨呈翼状竖起于背部。
  4. 假性肌肉肥大：触之坚韧，以腓肠肌最明显。
  5. 心肌损害：心律不齐，右胸前导联出现高R波，左胸前导联出现深Q波；心脏扩大、心瓣膜关闭不全。
  6. 病情发展到12岁时不能行走，需坐轮椅。晚期肌肉明显萎缩、腱反射消失，膝、肘、髋关节屈曲不能伸直，呼吸肌萎缩。多数因呼吸道感染、心力衰竭而死亡。
