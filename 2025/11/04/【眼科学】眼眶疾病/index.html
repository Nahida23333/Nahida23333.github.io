眼眶炎症

眼眶特异性炎症

特异性炎症： 是指由于明确的病原微生物引起的感染性炎症，如细菌、真菌引起的眼眶蜂窝织炎。

眼眶蜂窝织炎： 病原微生物感染引起的眶内软组织急性炎症，属于眼眶特异性炎症，发病急剧，严重者可因波及海绵窦而危及生命。

病因

  1. 多见于儿童。
  2. 病原体多为金黄色葡萄球菌、溶血性链球菌，儿童以流感嗜血杆菌多见。

临床表现

眶隔前蜂窝织炎

局限于眼睑，主要表现为眼睑充血水肿，疼痛不明显、瞳孔及视力多不受影响、眼球运动多不受限。

眶深部蜂窝织炎

  1. 眼睑高度肿胀。
  2. 结膜充血水肿
  3. 眼球运动障碍
  4. 视力严重下降
  5. 全身中毒症状，严重者可因波及海绵窦而危及生命。

治疗

  1. 立即给予全身足量抗生素控制炎症（经验用药→敏感抗生素）
  2. 细菌培养及药敏试验
  3. 寻找感染源
  4. 脓肿形成后切开引流
  5. 注意海绵窦炎等并发症

眼眶特发性炎症（IOI）

临床表现

  1. 肌炎
  2. 泪腺炎
  3. 巩膜周围炎和视神经周围炎
  4. 弥漫性炎症
  5. 眼眶炎性肿块

治疗

糖皮质激素、免疫抑制剂。

甲状腺眼病（TED）

一种由自身免疫反应引起的慢性多系统损害的疾病，与甲状腺疾病密切相关，也成为Graves眼病。

临床表现

  1. 眼睑征
  2. 眼睑及结膜充血水肿；
  3. 眼球突出
  4. 眼球运动障碍、复视和斜视；
  5. 角膜和眼表改变；
  6. 视神经病变；
  7. 眼压升高。

影像学表现

眼外肌梭形增粗。

诊断

  1. 典型的眼部表现及症状；
  2. 甲状腺功能或甲状腺相关抗体异常；
  3. 影像学表现

治疗

  1. 激素治疗
  2. 眼眶放射治疗
  3. 手术治疗
