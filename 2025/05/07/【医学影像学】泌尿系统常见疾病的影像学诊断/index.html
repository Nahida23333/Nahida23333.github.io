【医学影像学】泌尿系统常见疾病的影像学诊断

发表于2025-05-07|更新于2025-05-12|医学影像学
|浏览量:

先天性肾发育不良

先天性孤立肾（Solitary Kidney）

临床症状：常无

影像表现

  • IVP: 单侧肾盂输尿管显影，一侧不显影
  • CT: 一侧增大的肾脏，皮质、髓质结构正常，可有轻度旋转不良，肾门朝向腹侧

鉴别诊断

异位肾；一侧肾发育不良

影像学图解

1、左侧孤立肾

2、异位肾

3、异位肾

肾重复畸形（Renal Duplication）

临床症状：常无

影像学征象

 IVP: 重复的肾盂与输尿管
  CT: 可见重复的肾盂与输尿管，冠状重建显示清楚

影像学图解

1、左侧肾分为上下两部分，分别有肾盂和输尿管，上方的肾盂、肾盏较小

2、双侧肾盂输尿管重复畸形

肾融合畸形

临床表现：无或腹部包块

影像学征象

  • IVP
    • 两肾一极（多下极）在中线部位靠近/融合如马蹄状
    • 位置低，肾轴倒“八”字形
    • 肾旋转不良，肾门朝向前方
    • 可有肾积水和结石
  • CT: 马蹄肾峡部位于腹主动脉及下腔静脉前方，呈带状，内可见拉长的下肾盏

影像学图解

1、IVP显示：肾位置低，下极融合成峡部，肾轴从外上斜向内下，肾盂位于前方，肾盏指向后方

2、CT马蹄肾：两肾下极越主动脉前方融合

3、马蹄肾三维重建

4、马蹄肾磁共振检查

多囊肾（polycystic kidney）

临床表现

  • 婴儿型：肾体积增大，早期死亡
  • 成人型：30岁以后出现肾功能异常，可合并肝脏及胰腺的囊性改变

影像学表现

  • 平片：双侧肾影增大
  • IVP: 双侧肾盂肾盏拉长、变形，呈“蜘蛛足”样
  • CT平扫：双肾体积增大，肾内布满大小不一的类圆形水样密度灶
  • 增强：囊间的肾实质正常强化

鉴别诊断：多发性肾囊肿

影像学图解

1、双肾轮廓明显增大呈波浪状，呈多房囊性病灶

2、肾影增大，肾盂、肾盏受压、移位、拉长、变形和分离，呈“蜘蛛足”样改变，肾盏、肾盂无破坏

3、肾影增大，肾皮质内见多发类圆形水样密度区，无强化，其间肾实质见明显强化

4、多囊肝、多囊肾

肾囊肿

影像学表现

平片

多无异常，偶见钙化。尿路造影表现为肾盂肾盏弧形受压、伸长、移位、变形、边缘锐利

CT表现

  1. 肾内单发或多发圆形、类圆形占位
  2. 多为水样密度，密度均匀，如合并出血则密度增高
  3. 边缘光滑、清楚
  4. 增强扫描无强化
  5. 肾盂、肾盏受压、变形、移位，无破坏

影像学图解

1、右肾囊肿：右侧上盏弧形受压，右肾盂、肾盏下移

2、肾囊肿合并囊壁钙化

3、双肾多发囊肿：增强扫描，双肾皮质内多发类圆形未强化的水样密度区

4、右肾囊肿

5、多囊肾合并囊肿内出血

泌尿系结石

病理

  • 多种成分：草酸钙、磷酸钙、尿酸盐和胱氨酸盐等
  • 90%阳性结石，草酸盐结石占70-80％
  • 输尿管结石多继发于肾结石，膀胱结石可原发或继发

检查方法

  • 平片：阳性结石
  • 造影： 阴性结石 、结石定位、肾功能、泌尿系积液
  • CT：小结石检出、定位准确、肾功能

临床表现

腹痛（向下腹和会阴放射）、血尿及继发感染

影像学检查方法

平片

泌尿系区域高密度影，各自具有相对特征性

  1. 肾结石（renal calculus ）：分层、桑椹状、鹿角状、铸形
  2. 输尿管结石（ureteral calculus ）：长轴与输尿管走行一致
  3. 膀胱结石（ bladder calculus ）：耻骨联合上，位置变化（除憩室结石）

造影

充盈缺损影，边界清楚而别于肿瘤

CT

高密度影，CT值>100HU，不同于出血（出血CT值没有这么高）

输尿管结石

一般由肾结石下移而来，易停留于生理狭窄处。长轴与输尿管走行一致

尿路造影、CT可确定密影是否位于输尿管内

注意：与横突、骶椎重叠
别：淋巴结钙化、静脉石

膀胱结石

  • Ｘ线：结石横置于盆腔中线耻骨联合上方
  • 原发或继发，单发或多发，圆形、卵圆形，大小不等，边缘光滑或毛糙，密度可以不均匀
  • 可随体位变动而移动；阴性结石造影时表现为充盈缺损

影像学表现

  1. 直接征象：泌尿系行程上结节状高密度影
    • 阳性结石：X线平片可以显示的结石，90%
    • 阴性结石：少数结石难以显示，如尿酸盐结石
  2. 继发表现：肾盏、肾盂、输尿管扩张积水，肾盂肾盏显影延迟、肾实质萎缩

注意：与胆系结石、淋巴结钙化、盆腔内容物和静脉石鉴别。侧位片肾、输尿管结石与脊柱影重叠

影像学图解

1、肾结石

2、静脉肾盂造影：用于结石定位，了解有无积水，发现阴性结石及鉴别平片可疑的钙化影

3、左肾铸型结石

4、右肾盏、肾盂-输尿管移行处结石

5、CT检查：能确切发现位于肾盂和肾盏内的高密度结石

6、双肾多发结石

7、右侧输尿管上段结石，右侧肾盂肾盏扩张积液

8、左侧输尿管上段结石，左侧肾盂肾盏扩张积液

9、双肾、左侧输尿管多发结石

10、膀胱结石

泌尿系结核

临床表现： 20-40岁,尿频、尿急、血尿或脓尿；全身症状（低热、乏力和消瘦等），尿中结核杆菌

肾结核病理：早期：皮质→髓质病变→结核性脓肿→肾乳头以及肾盏→空洞→肾盏、肾盂黏膜破坏→全肾干酪样病变及脓腔（结核性脓肾）→肾功能丧失

受累顺序：肾皮质→髓质→肾盏、肾盂→输尿管、膀胱

若全肾钙化致肾功能丧失，称肾自截

检查方法

X线平片检查

  • 早期-无异常发现
  • 晚期-肾内钙化：全肾钙化形成肾自截

IVP检查：

  • 肾盂肾盏：肾小盏边缘不整、虫蚀样破坏→整个肾盏破坏、变形、消失→晚期波及全肾，肾功能严重受损，造影不显影
  • 肾实质：干酪坏死灶与肾盏相通→造影剂溢出肾盂肾盏外，呈团状、片状高密度影（TB性脓肿）→全肾钙化（肾自截）

CT检查

早期

  1. 肾实质内结核脓肿：平扫低密度，增强扫描边缘强化，中央坏死不强化
  2. 肾盂、肾盏早期破坏显示不佳

进展期

  1. 肾盏、肾盂扩张呈囊状低密度影
  2. 肾实质内低密度结核空洞，边缘不整、钙化，增强无强化，延时增强可有造影剂进入空洞，可见高密度液平
  3. 肾盂、输尿管壁增厚，范围较广；肾盂积水
  4. 肾实质内钙化，呈多点状或不规则高密度灶

晚期

病肾缩小，广泛钙化，肾自截

逆行尿路造影：肾盂肾盏扩大形成不规则空腔

影像学表现

肾结核

  • X线：可无异常，有时可见肾实质内云絮状或环状钙化，甚至全肾钙化
  • 静脉肾盂造影（IVP）：早期局限在肾实质，未侵入肾乳头不影响肾盏及肾功能时，IVP无阳性发现

影像学图解

1、病变侵入肾小盏，表现末端小盏杯口圆钝，模糊不整，呈虫蚀状改变

2、右肾钙化

3、右肾自截

4、左肾结核

5、IVP：左肾上盏变细、充盈不良；CT：左肾见钙化灶，增强扫描肾皮质内见一脓腔

肾癌和尿路上皮癌

肾细胞癌（Renal Cell Carcinoma）

病理

  1. 圆形或椭圆形，可呈分叶状；瘤内常有坏死、囊变及钙化
  2. 肾癌可直接侵犯肾V和下腔V，形成瘤栓；远处转移血行为主
  3. 病理类型：透明细胞癌（70%）、乳头状细胞癌（10-20%）、嫌色细胞癌（5-10%）、集合管癌（1%）、未分化癌（罕见）

临床表现

无痛性肉眼血尿、协腹部痛和腹部肿块

影像学表现

X线

  1. 肾影增大呈不规则分叶状，轮廓不清
  2. 肾轴旋转移位
  3. 肿瘤钙化　约5-15％的肾癌可见肿瘤钙化

静脉肾盂造影

按肿瘤大小不同，与肾盏、肾盂距离不等及肾盂受侵与否有关

  1. 肿瘤占位效应，肾盏受压、变形、移位
  2. 肿瘤侵蚀，肾盂、肾盏破坏，腔内充盈缺损
  3. 输尿管受压、移位
  4. 肾功能受损

CT

  1. 分叶或类圆形肿块位于肾实质内，大者致肾外形改变
  2. 平扫呈略低或中等密度与肾实质分界欠清
  3. 增强扫描，早期瘤实质 强化明显 ，稍后肾实质强化更明显，致与肿瘤界线变清晰
  4. 肿瘤常发生坏死液化呈混合密度，坏死液化区无强化
  5. 大的肿瘤常侵犯肾脂囊．少数有钙化
  6. 静脉瘤栓和淋巴结转移

MRI表现

  1. T1WI及T2WI均呈中等或混杂信号，坏死区T1WI呈低信号，T2WI呈高信号
  2. 假包膜显示率高，其T2WI、T1WI均呈低信号
  3. 静脉瘤栓显示好
  4. 余表现同CT

肾盂癌（Renal Pelvic Carcinoma）

移行细胞癌占80％，呈乳头状生长，可向下种植至输尿管和膀胱。40岁以上，男性多见，常见症状为间歇性无痛血尿、腹部肿块及腰酸痛。

影像学表现

Ｘ线平片

一般无阳性表现。 当肿瘤造成盂管交界处梗阻，形成明显肾盂积水时，可见肾影增大

静脉或逆行肾盂造影

肾盂内不规则充盈缺损，边缘可不光滑、位置固定；沿肾盂壁浸润生长，表现为肾盂肾盏轮廓毛糙或不规则狭窄与扩大

CT表现

  1. 肾窦内肿块，密度类似或低于肾实质但高于尿液
  2. 肿块周围肾窦脂肪受压并侵入邻近肾实质
  3. 肾盂或肾盏梗阻时，出现肾积水表现
  4. 增强检查：肾窦肿块仅有轻度强化，延时扫描，当残存肾盏肾盂明显强化时，可清楚显示肿瘤造成的充盈缺损

MRI表现

  1. 肿块T1WI及T2WI信号均较低
  2. 余表现同CT

膀胱肿瘤（Bladder Tumors）

95％以上来自移行上皮，少数为鳞癌和腺癌。发病年龄多为50岁以上，男性多于女性。易发生于膀胱三角区和两侧壁，表面凹凸不平，可有溃疡，少数可有钙化。常见症状为无痛性间歇肉眼血尿，尿频、 尿急、尿痛等；晚期可出现下腹部肿块，尿潴留

影像学表现

平片

肿瘤区有时可见钙化影

膀胱造影

见局部大小不等，形态各异的充盈缺损

CT表现

  1. 位于膀胱侧壁和三角区，可伴局部膀胱壁不规则增厚
  2. 膀胱壁突向腔内的结节、分叶及菜花状软组织灶
  3. 大小不等，病灶小者密度均匀，大者坏死、出血和钙化，部分壁外浸润侵犯
  4. 增强扫描：早期强化明显，延迟扫描呈腔内低密度充盈缺损

MRI表现

  1. 形态学表现与CT检查相仿
  2. T1WI肿瘤信号强度类似正常膀胱壁，T2WI多为中等信号，显著高于正常膀胱壁
  3. 增强扫描早期肿瘤强化显著高于正常膀胱壁
  4. 周围组织侵犯及淋巴结转移
