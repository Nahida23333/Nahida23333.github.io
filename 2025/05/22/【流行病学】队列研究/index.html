【流行病学】队列研究

发表于2025-05-22|更新于2025-05-22|流行病学
|浏览量:

概述

概念

  1. 队列研究：是将人群按是否暴露于某可疑因素及其暴露程度分为不同组，追踪其各组的结局，比较不同组之间结局频率的差异，从而判定暴露因素与结局之间有无因果关联及关联大小的一种观察性研究方法。
  2. 暴露：泛指能影响结局的各种因素，即研究对象接触过的某种待研究的物质、具备某种待研究的特征或行为。

特点

  1. 时间上是前瞻性的；
  2. 属于观察性研究：暴露不是研究者人为给予或随机分配的，研究结局也是在非干预情况下产生的。
  3. 研究对象按暴露与否进行分组
  4. 是从因到果的研究。

队列的分类

  1. 固定队列：是指观察对象都在某一时刻或一个短时期之内进入队列，之后不再加入新的成员，随访观察至观察期终止，观察对象很少或几乎没有因为所研究疾病等结局事件之外的其他原因退出，即在整个观察期内队列成员是相对固定的。
  2. 动态队列：在整个观察期内，原有的队列成员可以不断退出，新的观察对象可以随时进入，即整个观察期内队列成员不是固定的。

类型

  1. 前瞻性队列研究：是队列研究的基本形式，即在研究开始时，根据每个研究对象的暴露情况对研究对象进行分组，此时研究结局还没有出现，需要随访观察一段时间，收集每个研究对象研究结局发生情况的信息。
  2. 历史性队列研究/回顾性队列研究：是根据研究开始时研究者已掌握的有关研究对象在过去某个时点暴露情况的历史资料进行分组，研究的结局在研究开始时已经发生，不需要前瞻性观察。
  3. 历史前瞻性队列研究/双向性队列研究：历史性队列研究常常因为追踪的历史太短，结局还没有充分显现，需要继续对研究对象前瞻性随访观察一段时间，即在历史性队列研究的基础上继续进行前瞻性队列研究，此即历史前瞻性队列研究。
类型 优点 缺点
前瞻性队列研究 ①研究者可直接获取第一手资料，资料偏性比较小；②从因到果研究，可信性强；③可计算发病率 ①如需现察大量人群，则花费太大；②如疾病的潜伏期很长，则需要观察的时间很长，这些都会影响其可行性
历史性队列研究 ①节省时间、人力和物力，出结果快；②适宜于长诱导期和长潜伏期的疾病；③也经常用于具有特殊暴露的职业人群的研究 常缺乏影响暴露与疾病关系的混杂因素的资料，以至影响暴露组与未暴露组的可比性;因资料积累时未受到研究者的控制，所以内容上未必符合要求
历史前瞻性队列研究 ①具有上述两种研究的优点，在一定程度上弥补了它们的不足；②最适宜评价对人体健康具有短时间效应和长期作用的暴露因素的研究 同理

研究设计与实施

确定研究因素

暴露因素(exposure factor)：指研究对象曾经接触过某些因素，或具备某些特征，或处于某种状态。

确定研究结局

研究结局/结局变量：随访观察中预期出现的与暴露因素有关的结果，也就是研究者所希望追踪的事件或某些指标的变化。

确定研究现场与研究人群

选择研究现场的原则

  1. 人口相对稳定，便于随访
  2. 预期研究结局发生率较高；
  3. 有较好的组织管理体系；
  4. 研究能够获得当地政府重视、群众理解和支持的现场；
  5. 当地文化教育水平较高，医疗卫生条件较好，交通较便利。

研究人群

基本原则：必须是未患研究结局疾病的人群，而且在追踪观察期有可能发生所研究结局疾病。

暴露组的选择

  1. 职业人群：从暴露于研究因素的职业人群选择的暴露组
  2. 特殊暴露人群：由于某种原因对某因素有高暴露率的人群。（罕见暴露研究首选，包括职业人群做暴露组时，常用历史性队列研究，因对暴露的危害作用多半不是一开始就认识到的，一旦认识到了，大多都采取了防护措施以减少暴露，所以不易进行前瞻性队列研究）
  3. 一般人群：某行政区划或地理区域内的人群中暴露于所研究因素的人群作为暴露组
  4. 有组织的人群团体：此类人群便于有效率地收集随访资料，且他们的职业和经历往往相同，可增加其可比性。

对照人群的选择

  1. 内对照：如果在一般人群或有组织的人群团体中选择暴露于所研究因素的对象作为暴露组，那么其余非暴露者就可作为对照组，此即内对照。
  2. 外对照：当选择职业人群或暴露人群作为暴露组时，往往不能从这些人群中选出足够数量的具有可比性的对照，常需在该人群之外寻找对照组，这样选择的对照称为外对照。
  3. 总人口对照：即在采用职业人群或特殊暴露人群作为暴露组时，以该地区全人口的发病或死亡率作为对照。
  4. 多重对照：同时选取上述两种或两种以上形式的对照。

样本量的估计

影响样本量的因素有：①结局发生率 $p_0$ ；②暴露组的估计结局发生率 $p_1$ ；③统计学要求的显著性水平 $\alpha$ ；④把握度 $1-β$ 。

资料的收集和随访

基线资料

基线资料为每个对象在研究开始时的基本情况。包括人口学资料以及可能的混杂因素信息、暴露因素信息和结局指标信息。

随访

随访：通过定期的访问或检查获取研究对象预期结局事件发生的情况或观察结局指标的变化，同时收集有关暴露和混杂因素变化的资料。

目的

  1. 确定研究对象是否仍处于观察之中
  2. 确定研究人群中的结局事件
  3. 收集有关暴露和混杂因素的资料，以备分析时用

对象

  1. 所有被选定的研究对象，都应采用相同的方法同等地进行随访，并坚持追踪到观察终止期
  2. 有时还必须对失访者进行补访
  3. 未能追访到的，应尽量了解其原因，以便进行失访原因分析，估计有无产生偏倚

内容

  1. 随访内容应与基线资料一致，但收集的重点是结局变量，其具体项目视研究目的与研究设计而不同
  2. 制订调查表或检查记录表记录各种追踪观察项目，使用时贯彻始终，不得改变，直至观察终止

观察终点与终止时间

  1. 观察终点：是指研究对象出现了预期的研究结局。
  2. 观察终止时间：整个研究工作截止的时间

随访期与随访间隔

随访间隔：间隔与次数视具体研究的需要而定，如果观察时间较短，在观察终止时一次搜集资料即可，反之需多次随访；此外，要考虑结局变量本身的一些特征(如病程快慢等)

资料整理与分析

结局发生率的计算与比较

  1. 累计发病率（CI）：当研究人群的数量比较多，人口比较稳定（即固定队列），无论发病强度大小和观察时间长短，均可计算研究疾病的累积发病率，即以整个观察期内的发病人数初一观察开始时的人口数。
  2. 发病密度（ID）：以人时为单位计算出来的发病率带有瞬时频率性质，即表示在一定时间内发生某病新病例的速度，称为发病密度。
  3. 标化比：以全人口的发病（或死亡）率作为标准，计算出观察人群的预期发病（或死亡人数），再求得观察人群中实际发病（或死亡）人数与此预期发病（或死亡）人数之比，即标化发病（或死亡）比（SMR）。

$累计发病率=\frac{观察期内发病人数}{观察开始时的人口数}×k$
$发病密度=\frac{观察期内的发病人数}{观察人年数}×k$
$SMR=\frac{观察发病（或死亡）数}{预期发病（或死亡）数}=\frac{观察发病（或死亡）数}{全人口发病（或死亡）率×观察人数}$

关联强度的估计

  1. 相对危险度/率比/危险度比（RR）：暴露组与非暴露组的发病（或死亡）率之比，是反映暴露与发病（或死亡）关联强度的最常用指标。
  2. 归因危险度/率差/特异危险度/超额危险度（AR）：是暴露组发病（或死亡）率与对照组发病（或死亡）率相差的绝对值。
  3. 归因危险度百分比（AR%）/病因分值（EF）：是指暴露人群中归因于暴露的那部分发病（或死亡）率占全部发病（或死亡）率的百分比。
  4. 人群归因危险度（PAR）：总人群发病（或死亡）率中归因与暴露的那部分。
  5. 人群归因危险度百分比（PARP，PAR%）：/人群病因分值（PEF）：总人群发病（或死亡）率中归因于暴露的部分占全部发病（或死亡）率的百分比。

$I_1$ 和 $I_0$ 代表暴露组和非暴露组的发病（或死亡）率； $I_t$ 代表全人群的发病（或死亡）率。
$RR=\frac{I_1}{I_0}$
$AR=I_1-I_0=I_0(RR-1)$
$AR%=\frac{I_1-I_0}{I_1}×100%=\frac{RR-1}{RR}×100%$
$PAR=I_t-I_0$
$PAR%=\frac{I_t-I_0}{I_t}×100%$

RR 关联强度
0.9—1.0；1.0—1.1 无关联
0.7—0.8；1.2—1.4 弱
0.4—0.6；1.5—2.9 中
0.1—0.3；3.0—9.9 强
＜0.1；10— 很强

剂量效应关系(dose-response relationship)

是指暴露的剂量越大，其效应越大。意义：若存在剂量效应关系，则暴露作为病因的可能性就越大。

偏倚及其控制

选择偏倚

  1. 志愿者偏倚：志愿者与非志愿者在关心健康、注意饮食习惯、禁烟、禁酒及体育锻炼等方面存在系统差别，因而，志愿者被选为观察对象，而非志愿者落选，研究结果往往有选择偏倚。
  2. 失访偏倚：研究对象因迁移、外出、死于非终点疾病或拒绝继续参加观察而退出队列所引起的偏倚。如果暴露组和对照组的失访率相近，各组中失访者和未失访者的基本特征和结局发生率相似，则可以认为通过该研究获得的各组发病率可以反映该人群的实际情况，失访对研究结果没有影响；否则，暴露与结局之间的关系可能因失访而被歪曲，这种情况称为失访偏倚。

偏倚的控制：预防为主，贯穿设计-实施-分析阶段：①随机抽样；②提高应答率；③对历史资料记录不全时慎用；④提高随访的依从性。

信息偏倚

获取暴露、结局或其它信息时所出现的系统误差，常导致调查对象的错误分组，又称为错分偏倚(misclassification bias)，包括暴露错分和疾病错分以及暴露与疾病的联合错分。

控制措施：选择精确稳定的测量方法、调准仪器、严格操作规程、严格培训调查员、提高诊断技术、明确各项标准等

混杂偏倚

由于第三变量的作用，致使研究因素与结果的关联被歪曲，这个第三变量即混杂因素。

在队列研究中，如果暴露组和对照组在一些影响研究结果的主要特征（如年龄、性别等）上不一致，就会产生混杂偏倚。

优点与局限性

优点

  1. 资料可靠：研究对象的暴露资料的收集在结局发生之前，并且都是由研究者亲自观察得到或来自历史记录；
  2. 可以直接获得暴露组和对照组的发病（或死亡率），直接计算出RR和AR等反映暴露与疾病关联强度的指标。
  3. 因果时间顺序明确，加之偏倚较少，故检验病因假说的能力较强，可证实因果联系。
  4. 有助于了解人群疾病的自然史。
  5. 能对一种暴露因素所致的多种疾病同时进行观察，分析一种暴露与多种疾病的关系。

局限性

  1. 人力、物力、财力、时间耗费较多，不易实施。
  2. 不适用于发病率很低的疾病的病因研究。
  3. 随访时间长，易产生失访偏倚。
  4. 随访过程中未知变量引入人群，或人群中已知变量的变化等都可使结局收到影响，使分析复杂化。
