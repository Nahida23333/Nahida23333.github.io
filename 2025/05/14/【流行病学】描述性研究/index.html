【流行病学】描述性研究

发表于2025-05-14|更新于2025-05-21|流行病学
|浏览量:

概述

概念

描述性研究是指利用已有的资料或通过专题调查所获得的资料，描述疾病或健康状况的三间分布特征，进而获得病因线索，提出病因假设。

特点

  1. 以观察为主要研究手段，不对研究对象采取任何干预措施。
  2. 暴露因素不是随机分配的，且研究开始时一般不设立对照组。
  3. 暴露与结局的时序关系无法确定。

种类

  1. 个例调查/个案调查：是指到发病现场对新发病例的接触史、家族及周围人群的发病或健康状况以及可能与发病有关的环境因素进行调查。
  2. 病例报告：是对临床上某种罕见的单个病例或少数病例进行研究的主要形式，属于定性研究的范畴。
  3. 病例系列分析：是临床医生最为熟悉的临床研究方法，它是对一组相同疾病病人的临床资料进行整理、统计、分析并得出结论。
  4. 现况研究：在一个特定的时点或时期内，在特定范围的人群中，对某种（些）疾病或健康状况以及相关因素进行调查的一种方法。

现况研究

概述

概念

现况研究/横断面研究/患病率研究/现患研究：是指按照事先设计的要求，在某一特定人群中，应用普查或抽样调查等方法收集特定时间内某种疾病或健康状况及有关因素的资料，以描述资料的分布特征。

特点

  1. 在设计阶段一般不设立对照组；
  2. 特定时间：现况研究关注的是某一特定时点上或某一特定时期内某一群体暴露与疾病的状况或联系；
  3. 确定因果关系时收到限制；
  4. 对研究对象固有的暴露因素可以作因果推断（如性别、种族、血型、基因型等在疾病发生前就存在、且不会因是否患病而发生改变的因素）；
  5. 用现在的暴露（特征）来代替或估计过去情况是条件的；
  6. 定期重复进行可以获得发病率资料；
  7. 一般不用于病程较短的疾病。

研究类型

普查

概念

普查即全面调查，是指为了了解某人群健康状况或某疾病的患病率，或制定某生物学检验标准，在特定时间内对特定范围内（某一地区或具有某种特征）人群中每一成员所作的调查或检查。

目的

  1. 早期发现、早期诊断和早期治疗某些疾病
  2. 了解疾病和健康状况的分布
  3. 了解当地居民健康水平
  4. 了解人体各类生化指标的正常值范围。

适用条件

  1. 足够的人力、物力和财力；
  2. 所调查的疾病患病率较高，或错过一个病例代价大；
  3. 具备诊断设备，诊断方法简单易行，且具有较高的灵敏度和特异度；
  4. 具备有效的疾病治疗手段。

优点

  1. 能发现人群中的全部病例，全面描述疾病的分布和特征；
  2. 可以同时调查目标人群中多种疾病或多种健康状况的分布情况；
  3. 不存在抽样误差。

缺点

  1. 操作难度大：工作量大，费用较高，组织工作复杂，参加普查的工作人员多，调查质量不易控制；
  2. 调查内容有限，不适用于患病率很低和现场诊断技术比较复杂的疾病。

抽样调查

概念

抽样调查是指在特定的时点、特定范围的某人群总体中，按照一定的方法抽取一部分有代表性的个体组成样本进行调查分析，以此推论该人群总体某种疾病的患病率和某些特征的一种调查。

优点

  1. 省时省力
  2. 由于调查范围小而使调查精度较高

缺点

  1. 操作复杂：抽样调查的设计、实施与资料分析均比普查复杂；
  2. 重复及遗漏不易被发现；
  3. 不适用于变异较大的资料和需要普查普治的疾病；
  4. 样本的代表性较普查弱。

抽样方法

  1. 单纯随机抽样/简单随机抽样：是最简单、最基本的抽样方法。从总体的N个对象中，利用抽签或随机数字法抽取n个对象，构成一个样本，总体中每个对象被抽到的概率相等（n/N）。
  2. 系统抽样/机械抽样：按照一定的顺序，机械地每隔若干单位抽取一个单位的方法。
  3. 整群抽样：从总体中直接抽取若干群组作为观察单位组成样本。
  4. 分层抽样：先根据某种特征将总体分为若干次级总体（层），然后再从每一层内进行单纯随机抽样，组成一个样本，这种抽样方法称为随机抽样。
  5. 多阶段抽样：将不同抽样方法结合起来使用，把抽样过程分为不同阶段的方法。其实施过程为：先从总体抽取范围较大的单元，称为一级抽样单位，再从每个抽得的一级单位中抽取范围较小的二级单元……以此类推，最后抽取其中范围更小的单元作为调查单位。

各种抽样方法的比较

单纯随机抽样 系统抽样 整群抽样 分层抽样
优点 简单直观，其他抽样基础，计算简便 易理解，简便易行，容易得到按照比率分配的样本 便于组织，节省经费，容易控制调查质量 可独立分析层参数，灵活，信息利用好
缺点 大样本不使用，变异大时代表性差 周期性影响大 抽样误差大 层变量的选择
适用范围 总体不大的情况 按抽样顺序个体随机分布情况 群间差异小 层间差异大
抽样误差 大 较小 最大 小

偏倚及其控制

选择偏倚

选择偏倚：由于选择研究对象方法上的错误或缺陷使研究结果系统的偏离真实值。其实质是被选入的研究对象与没有被选者特征上的差异所造成，造成样本不能代表总体。

  • 无应答偏倚：调查对象不合作或因种种原因不能或不愿意参加，由于这些人的身体素质、暴露情况、患病情况、嗜好等可能与应答者不同，由此产生的偏倚称为无应答偏倚。
  • 选择性偏倚：在调查过程中，没有严格按照随机化原则抽样或主观选择研究对象，从而导致样本偏离总体的情况。
  • 幸存者偏倚：在现况研究中，调查对象均为幸存者，无法调查死亡的对象，因此不能全面反映实际情况，带有一定的局限性和片面性。

偏倚的控制：①严格遵照抽样方法要求，确保随机化原则；②提高研究对象的依从性和受检率；③调查对象的选择

信息偏倚

信息偏倚：是指在研究的实施阶段从研究对象获取研究所需信息时所产生的系统误差。其本质通常表现为研究对象的某种特征被错误分类（misclassification），在对疾病状态和暴露状态判断时都有可能发生。

  • 调查对象引起的偏倚
    • 报告偏倚/说谎偏倚：询问调查对象有关问题时，由于种种原因回答不准确从而引起偏倚；
    • 回忆偏倚：调查对象对过去的暴露史回忆不清，由其家属代替回忆，特别是健康的调查由于没有患病的经历，而容易将过去的暴露情况等遗忘，进而导致回忆偏倚。
  • 调查员偏倚：调查员有意识地调查具有某些特征的对象，而不重视或马虎调查其他不具备某些特征的对象，而导致调查偏倚。
  • 测量偏倚：指测量工具、检验方法不准确，检验技术操作不规范等，或工作粗心而导致测量偏倚。

偏倚的控制：①采用标准调查问卷和标准培训调查员；②问题具体化；③使用标准测量仪器(每次使用前进行校准)；④对敏感性问题(如酗酒、药物滥用、性活动等)，调查对象自己填写；⑤有些暴露指标可以采用生物标记物客观衡量；⑥盲法(降低主观倾向性)。

混杂偏倚

混杂偏倚：又称混杂（confounding），是指在流行病学研究中，由于一个或多个潜在的混杂因素（confounding factor）的影响，掩盖或夸大了研究因素与疾病（或事件）之间的联系，从而使两者之间的真正联系被错误地估计。

现况研究的优缺点

优点

  1. 研究结果有较强的推广意义
  2. 有来自同一群体的自然形成的同期对照组，结果具有可比性
  3. 可同时观察多种因素

缺点

  1. 难以确定先因后果的时序关系
  2. 不能获得发病率资料
  3. 研究对象可能处于临床前期而被误定为正常人

生态学研究

概念

是描述性研究的一种，它是以群体为基本单位收集和分析资料，在群体的水平上描述不同人群中某因素的暴露情况与某种疾病的频率，研究某种因素与某种疾病之间的关系。

特点

  1. 最基本的特征：群体为单位（国家、城市、学校）
  2. 可以初步探索群体中某因素暴露与疾病的关系
  3. 无法得知个体的暴露与效应间的关系
  4. 是探索病因线索的一种方法

类型

  1. 生态比较研究：观察不同人群或地区某种疾病或健康状况的分布，然后根据同一时期，不同地区或人群疾病或健康状况分布的差异，探索差异产生的原因，提出病因假设。
  2. 生态趋势研究：是连续观察平均暴露水平的变化（或者给予干预）和一个群体中某种疾病或健康状况频率变化和关系，了解其变化趋势，通过比较暴露水平变化前后疾病或健康状况频率的变化情况，判断该暴露与某种疾病或健康状况的联系。

优点

  1. 应用常规或现成资料来进行研究，省时省力
  2. 对病因未明的疾病可提供病因线索（最显著的优点）
  3. 适宜于个体的暴露剂量无法测量的情况
  4. 当研究的暴露因素在一个人群中变异范围很小时，宜采用多个人群比较的生态学研究
  5. 适合于对人群干预措施的评价
  6. 在疾病监测中，可估计某疾病发展的趋势

局限

  1. 生态学谬误（ecological fallacy）：由于生态学研究以各个不同情况的个体“集合”而成的群体为观察和分析的单位，以及存在混杂因素等原因而造成研究结果与真实情况不符。
  2. 混杂因素难以控制
  3. 难以确定因果关系：暴露与疾病测量的准确性；时序性
