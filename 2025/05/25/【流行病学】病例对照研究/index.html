【流行病学】病例对照研究

发表于2025-05-25|更新于2025-05-25|流行病学
|浏览量:

概述

基本概念

病例对照研究是按照有无所研究的疾病或某种卫生事件，将研究对象分为病例组和对照组，分别追溯其既往（发病或出现某种卫生事件前）所研究因素的暴露情况，并进行比较，以推测疾病与因素之间有无关联及关联强度大小的一种观察性研究。

主要特点

  1. 属于观察性研究；
  2. 设立对照；
  3. 由果推因；
  4. 论证强度：病例对照研究不难观察到由因到果的发展过程，一般而言不难证实暴露因素与疾病之间的因果联系。

基本原理

  1. 以确诊患某种特定疾病的病人作为病例；
  2. 以不患该病但具有可比性的个体作为对照；
  3. 通过询问、实验室检查或复查病史，搜集既往危险因素暴露史；
  4. 测量并比较两组各因素的暴露比例，经统计学检验该因素与疾病之间是否存在统计学关联。

研究类型

非匹配病例对照研究

对于病例和对照之间的关系不作限制和规定。

匹配病例对照研究

  1. 匹配/配比：所选择的对照在某些因素或特征上与病例保持一致。
  2. 匹配过头：如果将不起混杂作用的因素作为匹配变量进行匹配，企图使病例与对照尽可能一致，不仅会增加选择对照的难度和工作量，而且可能将与结局事件有关的因素匹配掉而丢失某些重要信息，这种情况称为匹配过头。

成组匹配病例对照研究/频数匹配病例对照研究

是指对照组具有某种或某些因素或特征者所占的比例与病例组一致或相近，即病例组和对照组之间某些因素和特征的分布一致或接近。

个体匹配病例对照研究

个体匹配是指以个体为单位使病例和对照在某种或某些因素或特征方面相同或相近。1个病例可以匹配一个对照，这种情况叫配对。

特点 匹配方法 不匹配方法
混杂因素控制 通过匹配关键变量，在设计阶段控制混杂因素 在分析阶段通过统计方法（如多变量回归）控制混杂因素
统计效率 样本量受限时可以提高效率，因为匹配减少了关键变量的变异 可以提高样本的多样性和代表性，但可能需要更大的样本量以维持统计效力
混杂 可以减少由于混杂因素引起的偏倚 混杂偏倚的风险可能更高
样本代表性 可能受限于病例可匹配的对照数量，降低总体样本的代表性 可以从更广泛的人群中选择病例和对照，提高了样本的代表性
适用情况 适用于罕见疾病、已知混杂因素强烈且难以调整的情况 适用于探索性研究、多因素分析以及当样本量和资源允许的情况
特点 频数匹配（成组匹配） 个体匹配
匹配精确度 在群体水平上匹配，如按年龄分组匹配，不保证每个对照与病例在个体水平完全匹配 每个病例与一个或多个对照在指定的变量上进行精确匹配
样本选择灵活性 选择灵活，容易找到符合分组特征的对照 较为困难，特别是在找到符合所有匹配标准的对照时
分析复杂度 相对简单，通常使用未配对的统计分析方法 较为复杂，需要使用配对的统计分析方法
过度匹配风险 较低，因为匹配是在群体水平上进行的 较高，尤其是当匹配变量多时，可能会匹配一些与疾病无关的特征
适用情况 适用于大规模研究，或当可用的匹配对照有限时 适用于病例数量较少或需要高度控制特定变量的研究

匹配法注意事项

  1. 慎重选择匹配因素：只匹配明确的混杂因素
  2. 可疑病因及中间变量不作为匹配因素
  3. 比例一般为1:1，不建议匹配过多，一般不超过1:4（研究效率增加缓慢且增加工作量）
  4. 避免匹配过度（overmatching）

匹配的目的

  1. 提高研究效率
  2. 控制混杂因素的干扰

病例的选择

选择原则

  1. 代表性：选择的病例应足以代表产生病例的靶人群（即产生这组病人的目标人群）中的全部病例；
  2. 诊断明确：必须对所研究疾病的诊断标准作出明确的规定，所有病例都应负荷严格的诊断标准；
  3. 可比性：病例组与对照组在年龄、性别等主要特征方面无明显差异。

病例的类型

  1. 新发病例（首选）：研究期间新发生并诊断的病例。由于病例患病的时间短，对有关暴露的回忆比较清楚，提供的信息较为准确可靠，且不受生存因素的影响，可避免因临床预后的不同而引起偏倚，研究对象的配合性也较好。
  2. 现患病例：研究人群中已存在某病的病人。收集时间短、费用相对低。但现患病例对暴露史的回忆因患病时间较长而易发生偏差，难以区分暴露和疾病的时间顺序，而且容易掺入疾病迁延及存活的因素。
  3. 死亡病例：研究时已死亡的病例。费用低、出结果快，但所获资料准确性差。

病例的来源

  1. 以医院为基础：方便可行、节省费用、合作性好，信息较完整、准确，可能使罕见病唯一选择，但易发生选择偏倚。
  2. 以社区为基础：代表性强，但病例获得比较困难，工作量和工作难度均较大。

对照的选择

选择原则

  1. 代表性：所选择的对照应能代表目标人群暴露的分布情况，最好是全人群的一个无偏样本，或是产生病例的靶人群中全体为患该病人群的一个随机样本。
  2. 可比性：除研究因素（暴露因素）之外，其他有关因素在病例组和对照组的分布应一致，如年龄、性别、居住地等。

对照的来源

  1. 同一或多个医疗机构中其他疾病的病人
  2. 社区人口或团体人群中非该病病例或健康人
  3. 病例的邻居或同一住宅区内的健康人或非该病病人
  4. 病例的配偶、同胞、亲戚、同学或同事等

估计样本量

查表法或公式法： $n=2\bar{p}\bar{q}{(z_{\alpha}+z_{\beta})}^2/{(p_1-p_0)}^2$

暴露与疾病相关性分析

四格表 $\chi^2$ 检验公式： $\chi^2=\frac{ {(ad-bc）}^2T}{m_1m_0n_1n_0}$
当四格表中一个格子的理论数＞1但＜5，且总例数＜40时，用校正 $\chi^2$ 检验。

关联强度分析

比值比（OR）：病例组某因素的暴露比值与对照组该因素的暴露比值之比，反映了病例组某因素的暴露比值为对照组的若干倍。
$OR=\frac{a/c}{b/d}=\frac{ad}{bc}$

  • $OR=1$ ：研究因素与疾病之间无关联；
  • $OR＞1$ ：研究因素与研究的疾病成“正”联系，数值愈大，该因素为危险因素的可能性愈大；
  • $OR＜1$ ：研究因素与研究的疾病成“负”联系，数值愈小，该因素为保护因素的可能性愈大。

$OR$ 可信区间的计算

95% $CI$ 不包括1（ $OR＞1$ 或 $OR＜1$ ）:如果进行多次病例对照研究，有95%的可能 $OR$ 不等于1，该项研究 $OR$ 不等于1并非抽样误差所致，有理由认为研究因素是研究疾病的危险因素或保护因素。

95% $CI$ 包括1：如果进行多次病例对照研究，可能有95%的研究其OR值等于1或接近1，即研究因素与研究疾病无关。

分级分析

分级分析：将不同暴露水平的资料由小到大或由大到小分成多个有序的暴露等级，不同水平的各级分别与无暴露或最低水平的暴露作比较，以分析暴露与其他卫生事件之间是否存在剂量反应关系。

分层分析

分层分析：把病例组和对照组按不同特征（一般为可疑的混杂因素）分为不同层次，再分别在每一层内分析暴露与疾病的关联程度，从而可以在一定程度上控制混杂因素对研究结果的影响。

多因素分析

偏倚及其控制

选择偏倚

  1. 入院率偏倚/伯克森偏倚：当利用医院病人作为病例和对照时，由于所选的对照仅是某种或某些疾病病人的一部分，而不是目标人群的随机样本，病例也只是该医院或某些医院的特定病例，而且由于医院的医疗条件、病人的居住地区及社会经济文化等多方面的因素的影响，病人对医院以及医院对病人都有一定的选择性，因此作为病例组的病例也不是全体病人的随机样本。
  2. 现患病例-新发病例偏倚/奈曼偏倚：病例对照研究中的研究对象如果选自现患病例，特别是病程较长的现患病例，所得到的暴露信息可能与存活有关而与发病无关，或者是由于疾病而改变了原有的一些暴露特征（如生活习惯），与新发病例所提供的暴露信息有所不同，其结果可能将存活因素等作为疾病发生的影响因素，夸大或缩小了研究因素和研究疾病的真实关系。
  3. 检出征候偏倚/暴露偏倚：某因素虽不是病因，但其存在有利于某些体征或症状出现，病人常因这些与疾病无关的症状而就医，从而提高了早期病例的检出率，致使过高地估计了暴露程度，而产生系统误差。
  • 偏倚的控制：①合理地选择病例与对照；②尽可能地从多家(类)医院选择病例与对照。

信息偏倚

非差异性信息偏倚

测量偏倚

差异性信息偏倚

  1. 回忆偏倚：由于研究对象对暴露史或既往史回忆的准确性和完整性存在系统误差而引起的偏倚。
  2. 代理者偏倚：研究的疾病时一种快速致死的疾病，或者疾病能够影响病人提供有效信息的能力。此时病例的信息代理人和对照的信息代理人提供的信息准确性也可能存在差异。
  3. 调查偏倚：调查者对病例与对照调查时，自觉或不自觉地采取不同的询问方式（方法、态度、广度、深度等）收集信息，产生的这种系统误差称诱导偏倚，对暴露情况及诊断结果划分发生错误则会引起错误分类偏倚，研究对象因某种原因有意报告而非真实信息将导致报告偏倚。
  • 错误分类偏倚根据是否均匀错分，分为差异性错误分类偏倚和非差异性错误分类偏倚。
  • 偏倚的控制：

混杂偏倚

  • 混杂因素：与暴露相关、是结局的影响因素、不是中间变量
  • 特点：不易识别、不易确定、不易完全消除
  • 控制
    • 设计阶段：匹配法、限制、随机法（病例对照研究中不采用，见于实验性研究）
    • 分析阶段：分层分析、多因素分析、其他（如标化）

优点与局限性

优点

  1. 适用于罕见的、潜伏期长的疾病研究
  2. 需要的样本量较小
  3. 适用于多种暴露因素与某一种疾病关联的研究，也可进行多种因素间交互作用的研究
  4. 不仅应用于病因的探讨，也可用于疾病预后、临床疗效、药物不良反应、疫苗免疫学效果的评价及影响因素的研究

局限性

  1. 需要的样本量大；
  2. 难以避免回忆偏倚、选择偏倚；
  3. 难以确定暴露与疾病的时间先后顺序；
  4. 难以充分而直接分析研究因素与疾病之间的关系。

病例对照研究衍生的其他研究类型

巢式病例对照研究

基本原理

  1. 基于队列研究，采用病例对照的分析方法；
  2. 前瞻性，按队列研究方式进行：选择一项队列，收集基线资料，采集所研究的生物学标志的组织或体液标本储存备用；

匹配

按病例进入队列的时间（始）、疾病出现时间（终）、性别与年龄等匹配条件，从同一队列选择1个或数个非病例作对照，抽取病例与对照的基线资料并检测收集的标本

资料处理

按匹配病例对照研究方法分析。

病例-队列研究

基本原理

  1. 研究开始时，在队列中随机选取一组样本作为对照组
  2. 观察结束时，队列中出现被研究疾病所有病例作病例组
  3. 与随机对照组进行比较
  • 这种研究模式，可同时研究几种疾病，不同疾病有不同病例组，但对照组都是同一组随机样本。

病例-交叉设计

  • 暴露与某急性事件有关
  • 比较相同研究对象暴露情况
    • 在急性事件发生前一段时间与未发生事件的某段时间
    • 如果暴露与少见的事件(或疾病)有关，那么刚好在事件发生前一段时间内的暴露频率应该高于更早时间内的暴露频率。
