【医学影像学】超声诊断学基础：成像基本原理与图像分析

发表于2025-05-28|更新于2025-05-28|医学影像学
|浏览量:

超声：振动频率大于20,000Hz的声波。

超声诊断学：利用超声波的物理特性和人体组织器官的声学特性相互作用而产生的信息，经处理后形成图像、曲线、波形，借此进行诊断的一种物理学检验方法。

原理

电能→换能器→声能→传到与反射→换能器→电能→主机→显示屏。

超声探头与扫查方式

  • 常规探头

  • 专用探头
    - 腔内探头
    - 术中探头

  • 探头：线阵型、扇型、凸弧型

发展历程与简要介绍

A型超声

  • 机理：以波纹变化反映回声情况
  • 特点：一维波形图，不直观
  • 用途：鉴别液性和实性包块，测距（临床上不常用）

M型超声

  • 机理：以单声束取样，获得活动界面回声再以满扫描方式展开；
  • 特点：一维图像——时间运动曲线图
  • 用途：分析心脏和大血管的运动幅度

B型超声

采用亮度调制方式来显示深度方向所有界面的反射回波信息。它以明暗不同的光点反映回声变化，强回声光点明亮，弱回声光点黑暗，按扫描线逐行显示随深度变换的回波信号，即构成一幅二维切面图象。

正常人体软组织内部回声排列：肾窦＞胎盘＞胰腺＞肝脏＞脾脏＞肾皮质＞皮下脂肪＞肾髓质＞静脉血＞胆汁和尿液。

液体与囊肿为无回声；结石和钙化为强回声，实性肿瘤可为低、等、高或混合回声。

D型（超声多普勒法）

  • 机理：利用Doppler原理对心血管内血流进行探测分析
  • 频谱多普勒（PW+CW）：以频谱曲线现实，检验血流动力学参数
  • 彩色多普勒血流显像（CDFI）
  • 多普勒效应：声源与接受体之间的相对运动引起声波频率发生改变的现象，频率的变化称为频移 $f_d$ 。
    • $f_d=f-f_0=V cos\theta f_0/c$
  • 临床意义
    • 判断血流的方向及性质（层流、湍流）
    • 在心血管的检测中，测定是否存在分流和返流，并定量估测
    • 判断血管是否狭窄或闭塞，是否有血栓形成
    • 检测肿瘤内血管，为鉴别肿瘤的良恶性提供参考。

新技术

  • 弹性成像：反映组织的弹性、软硬度；鉴别疾病的良恶性
  • 超声造影：利用与人体软组织间回声特性明显不同的外界物质（超声造影剂）注入体腔内、管道内或血管内，用常规或造影剂特异成像技术达到增强解剖显像甚至功能显像水平，提高超声诊断和鉴别诊断能力的技术。
    • 功能：了解血流灌注特点：①诊断与鉴别诊断；②引导肿瘤治疗；③疗效评估。

基本概念

  1. 纵向分辨力：分辨声束传播方向上两个目标的最小距离，与超声频率成正比。
  2. 横向分辨力：分辨声束轴线垂直方向上两个目标的最小距离，与声束宽度成反比。
  3. 声阻抗：介质密度与声波波速的乘积.
  4. 超声波的衰减：超声波传播过程中能量的损失，称为衰减。衰减与不同组织、散射、反射有关。不同组织的衰减速度不同：液性＜实性＜钙化＜含气脏器。

频率高，分辨好，穿透差；频率低，分辨低，穿透强。 对应的临床应用：检测浅表器官，采用高频探头；检测深部脏器，采用低频探头。

当传播中的声波遇到二种不同声阻抗物质形成的反射界面时，二种物质的声阻抗差越大，则反射越强，声像图表现为高或强回声；反之，反射越弱，声像图表现为低回声，如物质内部不存在声阻抗差，则无反射，声像图表现为无回声。

超声的生物学效应

  1. 热效应：声波传播中声能可转化为热能。
  2. 空化效应：可以产生局部高压、高温。

超声的安全性

  1. 热指数（TI）：＜1.0；胎儿＜0.4，眼＜0.2；
  2. 机械指数（MI）：＜1.0；胎儿＜0.3，眼＜0.1；

超声检查的优点和局限性

局限性

  1. 对骨骼和含气脏器显示差；
  2. 局部断面，未显示整体空间位置；
  3. 病灶过小或声阻抗性不大大大大大，难以分辨；
  4. 具有对操作人员的依赖性。
